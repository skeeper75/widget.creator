### 매칭 우선순위

1. **정확히 일치**: 상품명이나 카테고리명과 정확히 일치
2. **포함**: 키워드가 상품명이나 카테고리명에 포함
3. **부분 일치**: 키워드의 일부가 일치

## 카테고리 검색 알고리즘

```python
def search_categories(query: str, catalog: dict) -> list:
    """
    사용자 쿼리로 카테고리 검색
    """
    results = []

    for category in catalog["categories"]:
        # 1. 정확히 일치
        if query == category["displayName"]:
            results.append((category, 1.0))
            continue

        # 2. keywords 포함
        for keyword in category["keywords"]:
            if query in keyword or keyword in query:
                score = 0.9
                results.append((category, score))
                break

        # 3. path 포함
        for path_item in category.get("path", []):
            if query in path_item or path_item in query:
                score = 0.8
                results.append((category, score))
                break

    # 점수순 정렬
    results.sort(key=lambda x: x[1], reverse=True)
    return [cat for cat, score in results]
```
