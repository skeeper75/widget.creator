# 접지별 페이지 배열 (Page Layout Matrix)

접지 방식에 따른 페이지 배치 규칙과 다이어그램.

---

## 1. 십자접지 (Cross Fold)

### 1.1 십자접지 16P

가장 일반적인 책자 대지 형태.

**핵심 규칙**: 마주보는 페이지 합 = 17 (총페이지 + 1)

```
앞면 (Top-Bottom):
┌────┬────┬────┬────┐
│  4 │ 13 │ 16 │  1 │
├────┼────┼────┼────┤
│  5 │ 12 │  9 │  8 │
└────┴────┴────┴────┘

뒷면 (Top-Bottom):
┌────┬────┬────┬────┐
│  6 │ 11 │ 14 │  3 │
├────┼────┼────┼────┤
│  7 │ 10 │ 15 │  2 │
└────┴────┴────┴────┘

180° 회전 페이지: 4, 5, 13, 12, 6, 7, 11, 10
```

**접지 순서**:
1. 세로 중앙선 접기 (책등 방향)
2. 가로 중앙선 접기 (상단 방향)
3. 다시 세로 접기

### 1.2 십자접지 8P (직각접지)

**핵심 규칙**: 마주보는 페이지 합 = 9

```
앞면:
┌────┬────┐
│  4 │  5 │
├────┼────┤
│  1 │  8 │
└────┴────┘

뒷면:
┌────┬────┐
│  6 │  3 │
├────┼────┤
│  7 │  2 │
└────┴────┘
```

### 1.3 십자접지 32P

**핵심 규칙**: 마주보는 페이지 합 = 33

4회 접지 필요. 용지가 얇을 때만 가능.

## 2. 병렬접지 (Parallel Fold)

### 2.1 반접지 (Half Fold) - 4P

```
접기 전:
┌────────────┬────────────┐
│   P2 (뒤)  │   P3 (뒤)  │
├────────────┼────────────┤
│   P1 (앞)  │   P4 (앞)  │
└────────────┴────────────┘
        ↓ 접기
    ┌────────┐
    │ P1│P4 │ (표지)
    │ P2│P3 │ (내면)
    └────────┘
```

### 2.2 3단접지 (Tri-Fold / Letter Fold) - 6P

**⚠️ 중요: 안쪽 패널 2~3mm 작게!**

```
접기 전 (앞면):
┌─────────┬─────────┬─────────┐
│ 패널 A  │ 패널 B  │ 패널 C  │
│ (99mm)  │ (99mm)  │ (97mm)  │ ← 안쪽 패널 작게
│   P6    │   P1    │   P2    │
└─────────┴─────────┴─────────┘

접기 전 (뒷면):
┌─────────┬─────────┬─────────┐
│   P5    │   P4    │   P3    │
└─────────┴─────────┴─────────┘

접은 후:
   ┌───────┐
   │ P1    │ ← 표지
   │ P6 P2 │ ← 안쪽
   └───────┘
```

### 2.3 Z접지 (Z-Fold / Accordion 3단)

```
┌─────────┬─────────┬─────────┐
│ 패널 A  │ 패널 B  │ 패널 C  │
│   P1    │   P2    │   P3    │
└─────────┴─────────┴─────────┘
      ↓ Z자 형태로 접기
   _____
  /     \
 /   Z   \
/         \
```

페이지 순서가 연속적이라 지도, 안내도에 적합.

## 3. 롤접지 (Roll Fold)

### 3.1 롤접지 4단 (8P)

**⚠️ 핵심: 안쪽 패널 크기 점차 감소 (2~3mm씩)**

```
접기 전:
┌────────┬────────┬────────┬────────┐
│ 패널1  │ 패널2  │ 패널3  │ 패널4  │
│ (76mm) │ (74mm) │ (72mm) │ (70mm) │
│   P8   │   P1   │   P2   │   P3   │
└────────┴────────┴────────┴────────┘
   바깥쪽  ←────────────────→ 안쪽

뒷면:
┌────────┬────────┬────────┬────────┐
│   P5   │   P6   │   P7   │   P4   │
└────────┴────────┴────────┴────────┘

접는 순서:
1. 패널4를 패널3 위로 접기
2. 패널3+4를 패널2 위로 접기
3. 패널2+3+4를 패널1 위로 접기
```

### 3.2 롤접지 3단 (6P)

```
┌────────┬────────┬────────┐
│ 패널1  │ 패널2  │ 패널3  │
│ (100mm)│ (98mm) │ (96mm) │ ← 안쪽으로 갈수록 작게
└────────┴────────┴────────┘
```

## 4. 대문접지 (Gate Fold)

### 4.1 기본 대문접지 (6P)

```
접기 전:
┌─────────────┬───────────────────────┬─────────────┐
│    날개     │        중앙           │    날개     │
│  (1/4 폭)   │      (1/2 폭)        │  (1/4 폭)   │
│    P6       │        P1             │    P2       │
└─────────────┴───────────────────────┴─────────────┘

접은 후:
    ┌─────────────────┐
    │   날개│ │날개   │ ← 양쪽 날개가 중앙으로
    │   P6 │P1│ P2   │
    └─────────────────┘
```

고급 브로슈어에 적합.

### 4.2 이중대문접지 (8P)

대문접지 후 반으로 한 번 더 접기.

---

## 5. 병풍접지 (Accordion Fold)

### 5.1 병풍 4단 (8P)

```
┌────────┬────────┬────────┬────────┐
│        │        │        │        │
│   P1   │   P2   │   P3   │   P4   │
│        │        │        │        │
└────────┴────────┴────────┴────────┘
   ↑        ↓        ↑        ↓
  산접기   골접기   산접기   골접기
```

펼치면 한눈에 모든 패널 보임.

### 5.2 병풍 5단, 6단

동일한 패턴으로 확장. 홍보물, 연간 캘린더에 사용.

## 6. 제본 방식별 조판 순서

### 6.1 중철제본 (Saddle Stitch)

바깥 페이지부터 안쪽으로 끼워넣기 (Nested).

**16P 중철**:
```
Sheet 1: [16,1] / [2,15]   ← 가장 바깥
Sheet 2: [14,3] / [4,13]
Sheet 3: [12,5] / [6,11]
Sheet 4: [10,7] / [8,9]    ← 가장 안쪽
```

**24P 중철**:
```
페이지 순서: 24,1,2,23,22,3,4,21,20,5,6,19,18,7,8,17,16,9,10,15,14,11,12,13
```

### 6.2 무선제본 (Perfect Binding)

각 대지별로 독립적으로 조판 후 순차 적층.

**100P 무선 (16P×7대지)**:
```
대지1: 1~16P   (내부는 십자접지 16P 배열)
대지2: 17~32P
대지3: 33~48P
대지4: 49~64P
대지5: 65~80P
대지6: 81~96P
대지7: 97~112P (마지막 12P는 여백)
```

## 7. 회전 및 방향 주의사항

### 7.1 180° 회전 페이지

십자접지 16P에서 일부 페이지는 180° 회전해서 배치해야 함:
- 앞면: 4, 5, 13, 12
- 뒷면: 6, 7, 11, 10

### 7.2 용지 결 (Grain Direction)

- **종목 (Long Grain)**: 결이 세로 방향
- **횡목 (Short Grain)**: 결이 가로 방향
- 접지 방향과 결 방향이 같으면 접기 쉬움

### 7.3 Work & Turn 배치

```
Work & Turn:
┌──────────┬──────────┐
│  앞면A   │  앞면B   │
└──────────┴──────────┘
     ↓ 좌우 뒤집기
┌──────────┬──────────┐
│  뒷면B   │  뒷면A   │
└──────────┴──────────┘

한 판으로 앞뒤 인쇄 가능 → CTP 50% 절감
```

## 8. 크립(Creep) 보정 상세

### 8.1 크립 발생 원리

중철제본에서 종이를 겹쳐 접으면 안쪽 페이지가 바깥으로 밀려나옴.

```
측면도:
    ┌─────────────┐  ← 표지 (가장 바깥)
   ╱│             │╲
  ╱ │             │ ╲
 ╱  │             │  ╲ ← 안쪽 페이지가 밀려나옴
╱   │             │   ╲
```

### 8.2 크립 계산

```
총 크립량(mm) = (페이지수 ÷ 4 - 1) × 용지두께

예) 48P 중철, 80g 모조지 (0.10mm)
   = (48÷4 - 1) × 0.10
   = 11 × 0.10
   = 1.1mm
```

### 8.3 스프레드별 보정

```
스프레드 1 (바깥): 0.00mm
스프레드 2: 0.10mm
스프레드 3: 0.20mm
...
스프레드 12 (안쪽): 1.10mm

보정 방향: 안쪽 페이지일수록 바인딩 쪽으로 이동
```

### 8.4 적용 조건

| 조건 | 적용 |
|-----|------|
| 중철 40P 이상 | 필수 |
| 중철 24~36P | 권장 |
| 중철 16P 이하 | 불필요 |
| 무선제본 | 불필요 |
