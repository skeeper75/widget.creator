# 무선책자 (Perfect Binding Book)

상품코드: 006-0002

## 상품 개요

소프트커버 제본 책자. 내지를 순차 적층하여 책등에 접착제로 고정.
최소 24페이지부터 최대 300페이지까지 제작 가능.

## 옵션 구조 (4 Layer)

### Layer 1: 내지

| 옵션 | 타입 | 값 | 비고 |
|-----|-----|---|-----|
| 사이즈 | select | 148×210mm (A5), 210×297mm (A4) | |
| 용지 | select | #무선내지 참조 (10종) | |
| 인쇄 | select | 단면, 양면 | |
| 페이지 | range | 24~300P | 2P 단위 증가 |

**내지 용지 (#무선내지)**:
- 백색모조지 100g, 120g
- 아트지 100g, 120g
- 스노우지 100g, 120g
- 앙상블 100g, 130g
- 몽블랑 100g, 130g

### Layer 2: 표지

| 옵션 | 타입 | 값 | 비고 |
|-----|-----|---|-----|
| 용지 | select | #무선표지 참조 (6종) | |
| 인쇄 | select | 단면, 양면 | |
| 코팅 | select | 코팅없음, 무광코팅(단면), 유광코팅(단면) | |
| 투명커버 | toggle | 있음/없음 | +1,000원 |
| 면지 | toggle | 있음/없음 | +800원 |

**표지 용지 (#무선표지)**:
- 백색모조지 220g
- 아트지 250g, 300g
- 스노우지 250g, 300g
- 몽블랑 240g

### Layer 3: 제본

| 옵션 | 타입 | 값 | 비고 |
|-----|-----|---|-----|
| 방식 | fixed | 무선제본 | 고정 |
| 방향 | select | 좌철, 우철 | |
| 책등 | calculated | 자동계산 | 페이지×0.05+1mm |

**책등 두께 계산**:
```
책등(mm) = (페이지수 / 2) × 용지두께 + 1

용지두께 참고:
- 모조지 100g: 0.10mm
- 모조지 120g: 0.12mm
- 아트지 100g: 0.08mm
- 아트지 120g: 0.10mm
```

### Layer 4: 후가공

| 옵션 | 타입 | 값 | 비고 |
|-----|-----|---|-----|
| 박가공 | conditional | 없음, 있음 | 조건부 |
| 박크기 | dimension | 30~170mm | 박가공=있음 시 필수 |
| 박칼라 | select | 8종 | 박가공=있음 시 필수 |
| 형압 | conditional | 없음, 양각, 음각 | 조건부 |
| 형압크기 | dimension | 30~170mm | 형압≠없음 시 필수 |
| 포장 | select | 개별포장없음, 수축포장 | |

**박칼라**:
금유광, 은유광, 동박, 먹유광, 적박, 청박, 홀로그램박, 트윙클박

## 가격 계산

### 공식
```
최종가 = (내지비 + 표지비 + 제본비 + 후가공비) × 수량 × (1 - 할인율)

내지비 = 용지단가 × (페이지수/2) + 출력비
표지비 = 용지단가 + 출력비 + 코팅비 + 투명커버 + 면지
제본비 = 수량구간별 단가
후가공비 = 박가공비 + 형압비 + 포장비
```

## 파일 가이드

### 업로드 파일
- 내지: 단면 시 페이지별 PDF, 양면 시 스프레드 PDF
- 표지: 펼침면 PDF (앞표지 + 책등 + 뒤표지)

### 표지 파일 크기
```
가로 = (판형가로 × 2) + 책등두께 + (블리드 × 2)
세로 = 판형세로 + (블리드 × 2)

예) A5, 100P (책등 6mm), 블리드 3mm:
가로 = (148 × 2) + 6 + 6 = 308mm
세로 = 210 + 6 = 216mm
```
