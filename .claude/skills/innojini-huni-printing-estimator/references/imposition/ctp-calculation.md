# CTP 판수 계산 가이드

## 기본 공식

```
CTP 판수 = 대지수 × 색상수 × 인쇄면수

대지수 = ceil(총페이지 ÷ 대지크기)
색상수 = 1(먹), 2(2도), 4(CMYK)
인쇄면 = 2(양면), 1(Work&Turn)
```

## 상세 계산 단계

### Step 1: 대지 수 결정

| 대지크기 | 접지횟수 | 100P 책자 | 대지수 | 낭비 |
|---------|---------|----------|-------|-----|
| 4P | 1회 | 100÷4 | 25대지 | 0P |
| 8P | 2회 | 100÷8 | 13대지 | 4P |
| 16P | 3회 | 100÷16 | 7대지 | 12P |
| 32P | 4회 | 100÷32 | 4대지 | 28P |

**선택 기준**: 낭비 페이지 vs 조판 복잡도 트레이드오프

### Step 2: 색상별 판수

| 인쇄타입 | 설명 | 판수 |
|---------|-----|-----|
| 1도(먹) | 검정만 | 1판/면 |
| 2도 | 검정+별색 | 2판/면 |
| 4도(CMYK) | 풀컬러 | 4판/면 |
| 별색추가 | 금,은,형광 등 | +1판/색상 |

### Step 3: 인쇄 방식

| 방식 | 설명 | 판수 영향 |
|-----|-----|---------|
| 편면인쇄 | 앞/뒤 별도 인쇄 | ×2 |
| Work & Turn | 한 판으로 앞뒤 | ×1 (50% 절약) |
| Work & Tumble | 한 판, 회전 방향 다름 | ×1 (50% 절약) |

**Work & Turn 조건**:
- 앞뒤 동일 구성일 때
- 대지 절반으로 조판 가능할 때

## 계산 예시

### 예시 1: 무선책자 100P, 4도양면

```
내지:
├─ 총 페이지: 100P
├─ 대지 크기: 16P
├─ 대지 수: ceil(100/16) = 7대지
├─ 색상: 4도 = 4판
├─ 인쇄면: 양면 = 2
└─ CTP: 7 × 4 × 2 = 56판

표지:
├─ 페이지: 4P (앞표지+책등+뒤표지)
├─ 대지: 1대지
├─ 색상: 4도 = 4판
├─ 인쇄면: 양면 = 2
└─ CTP: 1 × 4 × 2 = 8판

총 CTP: 56 + 8 = 64판
```

### 예시 2: 중철책자 16P, 4도양면, Work&Turn

```
├─ 총 페이지: 16P
├─ 대지 크기: 16P
├─ 대지 수: 1대지
├─ 색상: 4도 = 4판
├─ Work&Turn: 1 (한 판으로 앞뒤)
└─ CTP: 1 × 4 × 1 = 4판
```

### 예시 3: 무선책자 200P, 4도양면 + 별색1도

```
내지:
├─ 대지 수: ceil(200/16) = 13대지
├─ 색상: 4도 + 별색1 = 5판
├─ 인쇄면: 양면 = 2
└─ CTP: 13 × 5 × 2 = 130판

표지: 1 × 5 × 2 = 10판

총 CTP: 130 + 10 = 140판
```

## 판수 최적화 전략

### 1. 대지 크기 선택
```
페이지 수 | 추천 대지 | 이유
---------|----------|-----
~16P     | 16P      | 1대지로 완성
17~32P   | 16P      | 2대지, 관리 용이
33~64P   | 16P      | 4대지까지 효율적
65~128P  | 16P/32P  | 상황에 따라 선택
129P+    | 32P      | 대지 수 최소화
```

### 2. Work & Turn 적용 시점
- 페이지 수가 대지 크기의 절반일 때 최적
- 예: 8P 내용 → 16P 대지로 Work&Turn = 판수 50% 절약

### 3. 별색 최소화
- 별색 1개 = CTP 전체 대지 × 면수 추가
- 200P 책자에 별색 1개 = +26판 (13대지 × 2)
- 가능하면 CMYK로 대체 검토

## 제본 방식별 주의사항

### 중철제본 (Saddle Stitch)
- 페이지 배치가 끼워넣기 방식
- 최대 64P 권장 (접지 두께 한계)
- 페이지 수 = 4의 배수 필수

### 무선제본 (Perfect Binding)
- 순차 적층 방식
- 최소 24P 이상 권장 (책등 필요)
- 페이지 수 = 2의 배수 필수

### PUR제본
- 무선과 동일하나 접착제 다름
- 180도 펼침 가능
- 내구성 높음, 비용 증가
