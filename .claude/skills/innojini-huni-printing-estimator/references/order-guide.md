## 2. 상품별 옵션 선택 순서

### 📚 무선책자

```
Step 1: 판형 선택
  └─ A5 (148×210mm) - 일반 책자
  └─ B5 (176×250mm) - 잡지형
  └─ A4 (210×297mm) - 보고서형

Step 2: 페이지 수
  └─ 최소 24P ~ 최대 300P
  └─ 2P 단위로 증가
  └─ 💡 페이지 수 → 책등 두께 결정

Step 3: 내지 용지
  └─ 모조지: 필기 가능, 가성비
  └─ 아트지: 사진 선명, 광택
  └─ 스노우지: 무광택, 고급
  └─ 고급지: 몽블랑, 앙상블

Step 4: 표지 용지
  └─ 250g~300g 권장
  └─ 내지보다 두꺼운 용지

Step 5: 표지 코팅
  └─ 무광: 고급스러움
  └─ 유광: 선명한 색감
  └─ 코팅없음: 필기 가능

Step 6: 후가공 (선택)
  └─ 박가공: 금박, 은박 등
  └─ 형압: 양각, 음각
```

### 🎴 프리미엄엽서

```
Step 1: 사이즈 선택
  └─ 100×150mm - 일반 엽서
  └─ A6 (105×148mm) - 표준
  └─ A5 (148×210mm) - 대형

Step 2: 용지 선택
  └─ 몽블랑: 부드러운 촉감
  └─ 아코팩: 도톰한 느낌
  └─ 매쉬멜로우: 포근한 질감

Step 3: 인쇄 면수
  └─ 단면: 한 면만 인쇄
  └─ 양면: 앞뒤 모두 인쇄

Step 4: 코팅
  └─ 무광/유광/없음
```

### 🏷️ 스티커

```
Step 1: 모양 선택
  └─ 사각형: 기본
  └─ 원형: 씰스티커
  └─ 칼선(도무송): 자유 모양

Step 2: 사이즈
  └─ 30×30mm ~ 70×100mm
  └─ 칼선: 최소 10×10mm

Step 3: 재질
  └─ 아트지: 광택, 일반
  └─ 모조지: 무광, 필기 가능
  └─ 투명 PET: 투명 스티커
  └─ 크라프트지: 친환경 느낌
```

## 3. 옵션별 시각적 차이 설명

### 코팅 차이
```
┌─────────────────────────────────────────┐
│  무광코팅        유광코팅        코팅없음   │
│  ┌───────┐    ┌───────┐    ┌───────┐  │
│  │  ░░░  │    │  ▓▓▓  │    │       │  │
│  │  ░░░  │    │  ▓▓▓  │    │       │  │
│  └───────┘    └───────┘    └───────┘  │
│   차분함         광택          자연스러움   │
│   고급스러움      선명한 색      필기 가능   │
└─────────────────────────────────────────┘
```

### 제본 방식 차이
```
중철제본 (Saddle Stitch)     무선제본 (Perfect Binding)
     ┌─────┐                      ┌─────┐
     │     │                      │     │
    /│     │\                     │     │▌ ← 책등
   / │     │ \                    │     │▌
  │  │     │  │                   │     │▌
  │  └─────┘  │                   └─────┘
  └───────────┘
  
  • 중앙 철심 제본              • 책등 접착 제본
  • 4~64P 적합                 • 24P 이상
  • 완전히 펼침 가능            • 책등에 제목 가능
```

### 용지 촉감
```
┌─────────────────────────────────────────┐
│  부드러움 ◄──────────────────► 까슬함    │
│                                         │
│  몽블랑 < 매쉬멜로우 < 스노우지 < 모조지  │
│                                         │
│  광택 ◄──────────────────────► 무광     │
│                                         │
│  아트지 < 스노우지 < 모조지 < 크라프트    │
└─────────────────────────────────────────┘
```

## 7. 자주 묻는 질문 (FAQ)

### Q: 페이지 수는 어떻게 결정하나요?
A: 콘텐츠 양에 따라 결정. 무선책자는 24P 이상, 중철은 4P 단위.

### Q: 용지는 직접 볼 수 있나요?
A: 후니프린팅 방문 상담 또는 전화(02-3409-8853)로 용지 샘플 요청 가능.

### Q: 코팅을 꼭 해야 하나요?
A: 필수 아님. 필기 필요하면 코팅없음, 내구성 원하면 코팅 추천.

### Q: 후가공은 나중에 추가할 수 있나요?
A: 파일 접수 전까지 옵션 변경 가능. 접수 후에는 불가.

### Q: 상담은 어떻게 받나요?
A: 전화 02-3409-8853 (영업시간: 10:00~18:00, 점심 12:30~13:30 제외)
