# 후가공별 파일 제작 가이드

후가공 옵션 선택 시 파일 제작 규칙입니다.
후가공 템플릿을 반드시 다운로드하여 확인하세요.

---

## 1. 코팅 (라미네이팅)

### 종류
| 코팅 | 특징 | 용도 |
|-----|-----|-----|
| 유광코팅 | 광택, 선명한 색감 | 사진, 카탈로그 |
| 무광코팅 | 차분함, 고급스러움 | 명함, 브로셔 |
| 벨벳코팅 | 부드러운 촉감 | 프리미엄 인쇄물 |

### 파일 준비
- 별도 레이어 불필요
- 기본 파일 그대로 제출
- 단, 코팅 후 접지 시 **크랙(갈라짐)** 주의

### 주의사항
```
⚠️ 코팅 후 후가공 순서
   코팅 → 오시 → 접지 (순서 지켜야 크랙 방지)
```

## 2. 박가공 (금박/은박/홀로그램박)

### 파일 구성
```
PDF 레이어 구조:
├─ 레이어 1: 인쇄용 이미지 (CMYK)
└─ 레이어 2: 박가공 영역 (별도 레이어)
```

### 박 레이어 제작 규칙
| 항목 | 규칙 |
|-----|-----|
| 색상 | **별색(Spot Color)** 또는 **K100** |
| 레이어명 | "박" 또는 "FOIL" |
| 최소 선폭 | 0.5mm 이상 |
| 최소 글자 | 8pt 이상 (고딕체 권장) |

### 박 종류
```
금유광, 금무광
은유광, 은무광
로즈골드박
홀로그램박
청박, 적박, 먹박
```

### 주의사항
- 박 영역은 **벡터**로 제작 (이미지 X)
- 너무 가는 선이나 작은 글씨 피하기
- 박 레이어와 인쇄 레이어 **정확히 정렬**

```
⚠️ 박 레이어가 없으면 가공 불가
   별색 설정 안 하면 인쇄로 처리됨
```

## 3. 형압가공 (엠보싱/디보싱)

### 종류
| 형압 | 설명 |
|-----|-----|
| 양각(엠보싱) | 볼록하게 튀어나옴 |
| 음각(디보싱) | 오목하게 들어감 |

### 파일 구성
```
PDF 레이어 구조:
├─ 레이어 1: 인쇄용 이미지
└─ 레이어 2: 형압 영역 (별도 레이어)
```

### 형압 레이어 제작 규칙
| 항목 | 규칙 |
|-----|-----|
| 색상 | **별색** 또는 **K100** |
| 레이어명 | "형압" 또는 "EMBOSS" |
| 최소 선폭 | 1mm 이상 |
| 최소 글자 | 10pt 이상 |

### 주의사항
- 형압은 박보다 **더 두꺼운 선** 필요
- 복잡한 패턴 피하기
- 종이 두께에 따라 효과 다름 (두꺼울수록 선명)

## 4. 오시 / 접지

### 오시란?
- 접는 부분에 **눌림선**을 넣어 깔끔하게 접히게 함
- 코팅 후 필수 (크랙 방지)

### 오시 위치 표시
```
오시선: 재단선 기준 위치 표시
예) 3단 접지 리플렛
    ├─ 1단: 99mm
    ├─ 2단: 100mm
    └─ 3단: 100mm (표지)
```

### 파일 표시 방법
| 방법 | 설명 |
|-----|-----|
| 가이드선 | 비인쇄 레이어에 위치 표시 |
| 주문 시 명시 | 옵션에서 오시 위치 입력 |

### 접지 종류
| 접지 | 접는 횟수 | 용도 |
|-----|---------|-----|
| 2단접지 | 1회 | 카드, 초대장 |
| 3단접지 | 2회 | 리플렛 |
| 4단접지 | 3회 | 안내문 |
| 병풍접지 | 여러 번 | 메뉴판 |
| 대문접지 | 안으로 접음 | 브로셔 |

### 크기 계산 (3단접지 예시)
```
펼침 크기: 297mm (A4 가로)
├─ 안쪽 면: 97mm (살짝 작게)
├─ 가운데: 100mm
└─ 바깥 면: 100mm

⚠️ 안쪽으로 접히는 면은 1~2mm 작게
```

## 5. 미싱 (절취선)

### 용도
- 쿠폰, 티켓의 뜯는 부분
- 설문지 회수용

### 파일 표시 방법
```
미싱선: 점선으로 표시
색상: K100 또는 별색
레이어: 별도 레이어 권장
```

### 미싱 규격
| 항목 | 규격 |
|-----|-----|
| 선 두께 | 0.3~0.5pt |
| 점선 패턴 | 2mm 선, 1mm 간격 |
| 안전거리 | 가장자리에서 5mm 이상 |

## 6. 타공

### 용도
- 링 제본용 구멍
- 걸이용 구멍
- 태그/라벨

### 파일 표시 방법
```
타공 위치: 원형으로 표시
색상: 별색 또는 K100
레이어: 별도 레이어
```

### 타공 규격
| 용도 | 구멍 크기 | 위치 |
|-----|---------|-----|
| 일반 타공 | 4~6mm | 주문 시 지정 |
| 링 제본 | 3mm | 여백에서 6mm |
| 걸이용 | 5mm | 상단 중앙 |

### 주의사항
- 타공 위치에 중요 내용 배치 금지
- 가장자리에서 **5mm 이상** 떨어뜨리기

## 7. 귀도리 (모서리 라운딩)

### 용도
- 명함, 카드의 둥근 모서리
- 안전성 (날카로운 모서리 제거)

### 규격
| 크기 | 용도 |
|-----|-----|
| R3 (3mm) | 약한 곡선 |
| R5 (5mm) | 일반적 |
| R10 (10mm) | 강한 곡선 |

### 파일 준비
- 별도 표시 불필요
- 주문 시 옵션 선택
- 단, **도련 포함** 파일 필수

```
⚠️ 귀도리 적용 시 모서리 디자인 주의
   곡선으로 잘려나가는 영역 고려
```

## 8. 도무송 (커스텀 모양 재단)

### 용도
- 특수 모양 스티커
- 모양 카드, 태그
- 패키지 전개도

### 파일 구성
```
PDF 레이어 구조:
├─ 레이어 1: 인쇄용 이미지
└─ 레이어 2: 도무송 칼선 (별도 레이어)
```

### 칼선 제작 규칙
| 항목 | 규칙 |
|-----|-----|
| 색상 | **별색(Spot)** - 인쇄 안됨 |
| 선 두께 | 0.25~0.3pt |
| 레이어명 | "칼선" 또는 "DIE CUT" |
| 모양 | **닫힌 패스** (열린 패스 X) |

### 칼선 종류
| 선 종류 | 용도 |
|--------|-----|
| 실선 | 완전 절단 |
| 점선 | 미싱 (절취선) |
| 일점쇄선 | 접는 선 |

### 주의사항
```
⚠️ 칼선은 반드시 벡터로 제작
⚠️ 열린 패스 금지 (끝점 연결)
⚠️ 칼선 레이어 = 비인쇄 레이어
```

### 용도
- 투명/색지 위에 흰색 표현
- 밝은 색상 밑칠

### 파일 구성
```
PDF 레이어 구조:
├─ 레이어 1: CMYK 인쇄
└─ 레이어 2: 화이트 영역 (별도 레이어)
```

### 화이트 레이어 제작
| 항목 | 규칙 |
|-----|-----|
| 색상 | **별색** - 이름 "White" |
| 레이어명 | "화이트" 또는 "WHITE" |
| 오버프린트 | OFF |

### 화이트 인쇄 방식
| 방식 | 설명 |
|-----|-----|
| 화이트 Only | 흰색만 인쇄 |
| 화이트 + CMYK | 흰색 위에 컬러 인쇄 |
| 화이트 언더 | 컬러 밑에 흰색 깔기 |

## 후가공 템플릿 활용

### 템플릿 다운로드
1. 상품 주문 페이지 접속
2. 후가공 옵션 선택

### 템플릿 확인 방법
```
1. AI/PDF 템플릿 열기
2. 칼선/가이드 레이어 확인
3. 작업 파일에 위치 맞추기
```

```
⚠️ 템플릿의 칼선/가이드는 참고용
⚠️ 제출 파일에서는 반드시 삭제
⚠️ 칼선 위치 변경 불가 (템플릿 그대로)
```
