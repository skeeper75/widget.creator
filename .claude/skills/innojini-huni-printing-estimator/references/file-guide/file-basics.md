## 1. 색상 모드 (CMYK)

### RGB vs CMYK 차이
| 구분 | RGB | CMYK |
|-----|-----|------|
| 용도 | 화면 표시 | 인쇄 |
| 색상 범위 | 넓음 (형광색 가능) | 좁음 |
| 결과 | 밝고 선명 | 차분하고 안정적 |

### 주의사항
- **모든 파일은 CMYK로 제출**
- RGB/PANTONE 색상은 자동 변환되어 **색상 차이 발생**
- 형광색, 네온색은 CMYK로 표현 불가
- 모니터와 실제 인쇄물은 색상 차이 있음

### 검은색 설정
| 용도 | 설정값 | 비고 |
|-----|-------|-----|
| 텍스트 | K100 (C0 M0 Y0 K100) | 선명한 검정 |
| 넓은 면적 | Rich Black (C60 M40 Y40 K100) | 깊은 검정 |

```
⚠️ K100만 사용하면 넓은 면적에서 회색빛으로 보일 수 있음
```

## 2. 해상도 (300dpi)

### 권장 해상도
| 인쇄 종류 | 해상도 | 비고 |
|---------|-------|-----|
| 일반 인쇄 | 300dpi | 표준 |
| 대형 실사 | 150dpi | 먼 거리 시청 |
| 현수막 | 72~150dpi | 원거리용 |

### 해상도 확인 방법
- Photoshop: 이미지 > 이미지 크기
- Illustrator: 문서 정보 > 이미지
- 인쇄 크기에서 확인 (화면 100% 아님)

```
⚠️ 저해상도 이미지는 인쇄 시 깨짐 현상 발생
   웹용 이미지(72dpi)를 확대해도 품질 향상 안됨
```

## 3. 도련(Bleed) / 재단물림

### 개념
```
┌─────────────────────────┐
│     도련 영역 (3mm)      │ ← 잘려나감
│  ┌───────────────────┐  │
│  │   재단선 (실제)    │  │
│  │  ┌─────────────┐  │  │
│  │  │ 안전영역    │  │  │ ← 중요 내용
│  │  │ (3mm 안쪽)  │  │  │
│  │  └─────────────┘  │  │
│  └───────────────────┘  │
└─────────────────────────┘
```

### 도련 설정값
| 영역 | 크기 | 설명 |
|-----|-----|-----|
| 도련(Bleed) | 사방 3mm | 재단 시 잘려나갈 여유 |
| 안전영역 | 재단선에서 3mm 안쪽 | 중요 내용 배치 |

### 왜 필요한가?
- 재단 과정에서 **오차(밀림)** 발생 가능
- 도련 없으면 가장자리에 **흰 여백** 생김
- 안전영역 벗어나면 **텍스트/로고 잘림**

### 작업 예시 (A4 기준)
```
실제 크기: 210 × 297 mm
작업 크기: 216 × 303 mm (도련 3mm 포함)
안전영역: 204 × 291 mm (재단선에서 3mm 안쪽)
```

## 4. 서체 아웃라인(곡선화)

### 왜 필요한가?
- 인쇄소에 해당 폰트가 없으면 **대체 폰트로 변경**
- 폰트 버전 차이로 **글자 깨짐** 발생

### 아웃라인 처리 방법
**Illustrator**
```
1. 전체 선택 (Ctrl+A)
2. 문자 > 윤곽선 만들기 (Ctrl+Shift+O)
3. 저장 전 반드시 확인
```

**InDesign**
```
1. 전체 선택
2. 문자 > 윤곽선 만들기
   또는 PDF 내보내기 시 포함 설정
```

**Photoshop**
```
1. 텍스트 레이어 선택
2. 레이어 > 래스터화 > 문자
   또는 레이어 병합
```

```
⚠️ 아웃라인 처리 후에는 텍스트 수정 불가
   원본 파일은 별도 보관하세요
```

```
- 도련: 포함
- 색상: CMYK
- 해상도: 300dpi
```

## 6. 별색(Spot Color) 처리

### 별색이란?
- CMYK 외 특수 잉크 (금색, 은색, 형광색 등)
- 스와치에 저장된 사용자 정의 색상

### 왜 삭제해야 하나?
- 디지털 인쇄는 **CMYK 4색만 사용**
- 별색은 자동 변환되어 **예상과 다른 색상** 출력
- 분판 오류로 **인쇄 품질 저하**

### 별색 삭제 방법 (Illustrator)
```
1. 스와치 패널 열기 (Window > Swatches)
2. 별색 확인 (아이콘에 점이 있으면 별색)
3. 별색 더블클릭 > Color Type: Process Color로 변경
4. 또는 Edit > Edit Colors > Convert to CMYK
```

## 8. 양면 인쇄 파일

### 파일 순서
```
앞1, 뒤1, 앞2, 뒤2, 앞3, 뒤3 순서로 저장
```

### 양면 방향
| 제본 방향 | 설명 |
|---------|-----|
| 좌철 (왼쪽 제본) | 일반적인 책 방향 |
| 우철 (오른쪽 제본) | 아시아 전통 서적 |
| 상철 (위쪽 제본) | 달력, 메모장 |

### 펼침면 작업 시
```
중앙 이미지가 연결되려면:
- 왼쪽 페이지: 오른쪽으로 5mm 여유
- 오른쪽 페이지: 왼쪽으로 5mm 여유
→ 총 10mm 겹침 영역 확보
```

## 9. 이미지 링크 처리

### 링크 vs 포함
| 구분 | 링크 | 포함(Embed) |
|-----|-----|------------|
| 파일 크기 | 작음 | 큼 |
| 이동성 | 낮음 (이미지 분실 위험) | 높음 |
| 인쇄용 | 비권장 | **권장** |

### 이미지 포함 방법 (Illustrator)
```
1. Window > Links 패널 열기
2. 링크된 이미지 선택
3. 패널 메뉴 > Embed Image(s)
```
