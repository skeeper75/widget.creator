# 상품별 파일 규격 가이드

상품 유형별 파일 제작 규격입니다.
재단 크기 + 도련 3mm = 작업 크기

---

## 1. 명함

### 표준 규격
| 사이즈 | 재단 크기 | 작업 크기 (도련 포함) |
|-------|---------|-------------------|
| 일반명함 | 90 × 50mm | 96 × 56mm |
| 미니명함 | 85 × 45mm | 91 × 51mm |
| 유럽명함 | 85 × 55mm | 91 × 61mm |

### 파일 규격
```
해상도: 300dpi
색상: CMYK
도련: 사방 3mm
안전영역: 재단선에서 3mm 안쪽
서체: 아웃라인 필수
```

### 주의사항
- 텍스트 최소 크기: 6pt 이상
- 얇은 선: 0.3pt 이상
- 양면 인쇄 시: 앞면, 뒷면 순서로 저장

## 5. 책자

### 중철제본 (Saddle Stitch)
```
페이지 수: 4의 배수 (8P, 12P, 16P, 24P...)
최대: 64P 권장

파일 구성:
├─ 표지 파일 (표1-표4 펼침)
└─ 내지 파일 (페이지 순서대로)

또는 전체 1개 파일:
1P(표1), 2P(표2), 3P(내지1), 4P(내지2)...
```

### 무선제본 (Perfect Binding)
```
페이지 수: 2의 배수 (24P 이상 권장)
최소: 24P (책등 형성 필요)

파일 구성:
├─ 표지 파일 (펼침면: 뒤표지 + 책등 + 앞표지)
└─ 내지 파일 (페이지 순서대로)
```

### 표지 파일 크기 계산 (무선제본)
```
가로 = (판형 가로 × 2) + 책등 + (도련 × 2)
세로 = 판형 세로 + (도련 × 2)

예) A5 무선책자 100P (책등 6mm)
가로 = (148 × 2) + 6 + (3 × 2) = 308mm
세로 = 210 + (3 × 2) = 216mm
```

### 책등 두께 계산
```
책등(mm) = (페이지수 ÷ 2) × 용지두께

용지별 두께 (참고값):
├─ 모조지 80g: 0.10mm
├─ 모조지 100g: 0.12mm
├─ 모조지 120g: 0.14mm
├─ 아트지 100g: 0.08mm
├─ 아트지 120g: 0.10mm
└─ 아트지 150g: 0.12mm
```

|-------|---------|---------|
| 원형 30mm | Ø30mm | Ø36mm |
| 원형 50mm | Ø50mm | Ø56mm |
| 정사각 50mm | 50 × 50mm | 56 × 56mm |
| 직사각 30×50 | 30 × 50mm | 36 × 56mm |

### 도무송 스티커
```
파일 구성:
├─ 레이어 1: 인쇄 이미지 (CMYK)
└─ 레이어 2: 칼선 (별색, 0.25pt)

칼선 규칙:
- 닫힌 패스 필수
- 최소 크기: 10 × 10mm
- 칼선 간격: 최소 2mm
```

### 칼선 vs 인쇄 영역
```
┌─────────────────┐
│    도련 3mm     │
│  ┌───────────┐  │
│  │ 칼선(재단)│  │
│  │ ┌───────┐ │  │
│  │ │인쇄영역│ │  │
│  │ └───────┘ │  │
│  └───────────┘  │
└─────────────────┘

인쇄는 칼선보다 2mm 안쪽까지
```

## 파일 용량 가이드

### 권장 용량
| 상품 | 권장 용량 | 최대 용량 |
|-----|---------|---------|
| 명함 | ~10MB | 50MB |
| 엽서/카드 | ~20MB | 100MB |
| 리플렛 | ~50MB | 200MB |
| 책자 | ~100MB | 500MB |
| 포스터 (A1+) | ~200MB | 1GB |

### 용량 초과 시
1. 이미지 해상도 확인 (300dpi 이상 불필요)
2. 불필요한 레이어 병합
3. PDF 최적화 저장
4. 이미지 압축 (품질 80% 유지)
