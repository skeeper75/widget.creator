# 크립(Creep) 보정 레퍼런스

## 개요

중철제본 책자에서 안쪽 페이지가 바깥쪽으로 밀려나오는 현상을 크립(Creep)이라고 합니다.
이를 보정하지 않으면 안쪽 페이지의 여백이 좁아지거나 내용이 잘릴 수 있습니다.

---

## 1. 크립 발생 원리

```
중철제본 책을 옆에서 본 모습:

        ┌─────────────┐ ← 표지
       ╱│             │╲
      ╱ │             │ ╲
     ╱  │             │  ╲ ← 안쪽 페이지가
    ╱   │             │   ╲   바깥으로 밀림
   ╱    │             │    ╲
  ╱     │             │     ╲
 ╱      │             │      ╲
╱───────│─────────────│───────╲
        │   (책등)    │

안쪽 페이지일수록 더 많이 밀려나옴
```

### 왜 밀려나는가?
- 종이를 여러 장 겹쳐서 접으면 안쪽 종이의 둘레가 더 길어짐
- 바깥쪽 종이는 더 큰 호를 그리며 접힘
- 결과적으로 안쪽 종이가 바깥으로 튀어나옴

---

## 2. 적용 조건

### 필수 조건
| 조건 | 값 |
|-----|---|
| 제본 방식 | **중철제본만** |
| 페이지 수 | **40P 이상** |

### 무선제본은?
무선제본은 페이지를 쌓아서 책등에 접착하므로 크립이 발생하지 않습니다.

### 40P 미만은?
페이지가 적으면 크립량이 미미하여 보정 불필요합니다.

---

## 3. 계산 공식

### 총 크립량
```
총_크립량(mm) = (페이지수 ÷ 4 - 1) × 용지두께
```

### 스프레드별 보정값
```
스프레드 = 마주보는 두 페이지 (예: 2-3, 4-5, ...)
총_스프레드 = 페이지수 ÷ 4

스프레드별_보정 = 총_크립량 × (스프레드_번호 - 1) / (총_스프레드 - 1)
```

## 4. 계산 예시

### 예시 1: 48P 중철, 80g 용지

```
용지 두께: 0.10mm (80g)
총 스프레드: 48 ÷ 4 = 12
총 크립량: (12 - 1) × 0.10 = 1.1mm

스프레드별 보정값:
  스프레드  1 (P1-2, P47-48):  0.00mm (가장 바깥)
  스프레드  2 (P3-4, P45-46):  0.10mm
  스프레드  3 (P5-6, P43-44):  0.20mm
  스프레드  4 (P7-8, P41-42):  0.30mm
  스프레드  5 (P9-10, P39-40): 0.40mm
  스프레드  6 (P11-12, P37-38): 0.50mm
  스프레드  7 (P13-14, P35-36): 0.60mm
  스프레드  8 (P15-16, P33-34): 0.70mm
  스프레드  9 (P17-18, P31-32): 0.80mm
  스프레드 10 (P19-20, P29-30): 0.90mm
  스프레드 11 (P21-22, P27-28): 1.00mm
  스프레드 12 (P23-24, P25-26): 1.10mm (가장 안쪽)
```

### 예시 2: 64P 중철, 100g 용지

```
용지 두께: 0.11mm (100g)
총 스프레드: 64 ÷ 4 = 16
총 크립량: (16 - 1) × 0.11 = 1.65mm

스프레드 1: 0.00mm
스프레드 8: 0.77mm
스프레드 16: 1.65mm
```

## 5. 보정 적용 방법

### InDesign에서
1. 문서 설정 → 페이지 크기
2. 크립 보정값만큼 내부 여백을 조정
3. 또는 자동 크립 조정 기능 사용

### 수동 조정
```
안쪽 여백 조정값 = 기본 여백 - (크립 보정값 ÷ 2)

예) 기본 내부 여백 15mm, 크립 보정 1mm:
  조정 후 내부 여백 = 15 - 0.5 = 14.5mm
```

### 시각적 다이어그램

```
보정 전 (크립 발생):
┌─────────────────────────────────┐
│  ┌─────────┐    ┌─────────┐    │
│  │ 넓은   │    │ 좁은   ←│ 잘림│
│  │ 여백   │    │ 여백    │    │
│  └─────────┘    └─────────┘    │
│   바깥 페이지     안쪽 페이지    │
└─────────────────────────────────┘

보정 후 (균일한 여백):
┌─────────────────────────────────┐
│  ┌─────────┐    ┌─────────┐    │
│  │ 조정된 │    │ 조정된 │    │
│  │ 여백   │    │ 여백   │    │
│  └─────────┘    └─────────┘    │
│   바깥 페이지     안쪽 페이지    │
└─────────────────────────────────┘
```

## 6. CLI 사용법

```bash
# 크립 보정 계산
python3 scripts/estimate_v3.py creep --pages 48 --paper-weight 80

# JSON 출력
python3 scripts/estimate_v3.py creep --pages 48 --paper-weight 80 --json

# 불필요한 경우 (32P)
python3 scripts/estimate_v3.py creep --pages 32 --paper-weight 80
# → "크립 보정이 필요하지 않습니다 (40P 미만)"
```

## 7. 주의사항

1. **무선제본 불필요**: 무선제본은 크립이 발생하지 않음
2. **40P 미만 불필요**: 크립량이 미미하여 보정 생략 가능
3. **두꺼운 용지 주의**: 평량이 높을수록 크립량 증가
4. **인쇄소 협의**: 실제 보정은 인쇄소와 협의 필요
