# 표지 오시 계산 레퍼런스

## 개요

무선제본/양장제본 표지의 오시(접힘선) 위치와 개수를 계산하는 방법을 정리합니다.

---

## 1. 책등 두께 계산

### 공식
```
책등두께(mm) = (페이지수 ÷ 2) × 용지두께
```

### 용지 두께 테이블

| 평량(g) | 두께(mm) | 비고 |
|--------|---------|------|
| 60 | 0.08 | 신문용지 |
| 70 | 0.09 | 서적용지 |
| 80 | 0.10 | 모조지 (기본) |
| 100 | 0.11 | 모조지 |
| 120 | 0.12 | 모조지/아트지 |
| 150 | 0.14 | 고급지 |
| 180 | 0.16 | 두꺼운 내지 |
| 200 | 0.18 | 표지용 |
| 250 | 0.22 | 표지용 |
| 300 | 0.26 | 표지용 |
| 350 | 0.30 | 하드커버 |

### 계산 예시

```
예) 200P 책, 내지 80g
  책등 = 200 ÷ 2 × 0.10 = 10.0mm

예) 300P 책, 내지 100g
  책등 = 300 ÷ 2 × 0.11 = 16.5mm
```

## 2. 오시 줄 수

### 제본 방식별 오시

| 제본방식 | 날개 | 오시 줄 수 | 구성 |
|---------|-----|----------|------|
| **무선제본** | 없음 | 4줄 | 책등 2줄 + 힌지 2줄 |
| **무선제본** | 있음 | 6줄 | 책등 2줄 + 힌지 2줄 + 날개 2줄 |
| **중철제본** | - | 0줄 | 오시 불필요 (반으로 접음) |
| **양장제본** | - | 4줄 | 힌지 4줄 (책등은 별도 보드) |

### 오시 위치 다이어그램 (무선제본, 날개 있음)

```
←───────────────── 표지 펼침 ─────────────────→

┌────────┬─────────┬─────┬─────────┬────────┐
│        │         │     │         │        │
│  날개  │  뒤표지  │책등 │  앞표지  │  날개  │
│        │         │     │         │        │
└────────┴─────────┴─────┴─────────┴────────┘
    ↑         ↑     ↑ ↑     ↑         ↑
   오시6     오시4  2  1   오시3     오시5
   (날개)   (힌지) 책등  (힌지)   (날개)

오시 위치:
  1. 책등 왼쪽 (앞표지-책등 경계)
  2. 책등 오른쪽 (책등-뒤표지 경계)
  3. 앞표지-날개 경계 (힌지에서 6~10mm)
  4. 뒤표지-날개 경계 (힌지에서 6~10mm)
  5. 앞날개 끝 (접히는 부분)
  6. 뒷날개 끝 (접히는 부분)
```

## 3. 힌지 간격

### 평량별 힌지 간격

| 표지 평량 | 힌지 간격 | 비고 |
|----------|----------|------|
| 250g 이하 | 6mm | 일반 표지 |
| 250~300g | 8mm | 두꺼운 표지 |
| 300g 초과 | 10mm | 매우 두꺼운 표지 |

### 힌지의 역할

힌지는 책을 펼쳤을 때 표지가 자연스럽게 열리도록 하는 오시입니다.
표지가 두꺼울수록 힌지 간격을 넓혀야 책이 잘 펼쳐집니다.

```
┌─────────────────────────────────────┐
│                                     │
│  ┌───────────┬─────┬───────────┐   │
│  │           │     │           │   │
│  │  뒤표지   │책등 │  앞표지   │   │
│  │           │     │           │   │
│  └───────────┴─────┴───────────┘   │
│       ↑    ↑         ↑    ↑        │
│       힌지           힌지          │
│       간격           간격          │
│    (6~10mm)       (6~10mm)         │
│                                     │
└─────────────────────────────────────┘
```

## 4. 표지 펼침 크기 계산

### 무선제본 (날개 없음)

```
가로 = 도련 + 완성물너비 + 책등 + 완성물너비 + 도련
세로 = 도련 + 완성물높이 + 도련

예) A5 (148×210mm), 200P, 80g, 도련 3mm
  책등 = 200 ÷ 2 × 0.10 = 10mm
  가로 = 3 + 148 + 10 + 148 + 3 = 312mm
  세로 = 3 + 210 + 3 = 216mm
```

### 무선제본 (날개 있음)

```
가로 = 날개 + 도련 + 완성물너비 + 책등 + 완성물너비 + 도련 + 날개
세로 = 도련 + 완성물높이 + 도련

예) A5 (148×210mm), 200P, 80g, 날개 80mm, 도련 3mm
  책등 = 200 ÷ 2 × 0.10 = 10mm
  가로 = 80 + 3 + 148 + 10 + 148 + 3 + 80 = 472mm
  세로 = 3 + 210 + 3 = 216mm
```

## 5. CLI 사용법

```bash
# 기본 (날개 없음)
python3 scripts/estimate_v3.py cover \
  --width 148 --height 210 \
  --pages 200 --binding 무선

# 날개 포함
python3 scripts/estimate_v3.py cover \
  --width 148 --height 210 \
  --pages 200 --binding 무선 \
  --wing --wing-width 80

# JSON 출력
python3 scripts/estimate_v3.py cover \
  --width 148 --height 210 \
  --pages 200 --binding 무선 \
  --wing --wing-width 80 --json
```

### 출력 예시

```
📏 표지 오시 계산 결과

완성물: 148 × 210 mm
페이지: 200P
내지: 80g
표지: 250g
제본: 무선 (날개 80mm)

책등 두께: 10.0 mm
펼침 크기: 472.0 × 216.0 mm
오시 줄 수: 6줄 (책등 2 + 힌지 2 + 날개 2)
힌지 간격: 6.0 mm
```

---

## 6. 주의사항

1. **책등 최소 두께**: 무선제본은 최소 3mm 이상 필요
2. **힌지 위치**: 책등 가장자리에서 정확히 힌지 간격만큼 떨어진 위치
3. **날개 크기**: 보통 80~100mm, 최대 완성물 너비의 2/3
4. **도련**: 표준 3mm, 재단 여유
