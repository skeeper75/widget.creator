## 1. 텀블러 (Tumbler) 🥤

### 시장 특성

| 항목 | 내용 |
|------|------|
| **시장 규모** | 성장 중 (Stanley 열풍 이후) |
| **경쟁 강도** | 🔴 높음 |
| **마진** | 30-50% |
| **시즌성** | 여름 피크, 연중 판매 |

### 성공 전략

```
✅ DO:
- 니치 타겟팅 (운동, 캠핑, 오피스)
- 커스터마이징 옵션 (이름 각인)
- 번들 판매 (텀블러 + 빨대 + 브러시)
- 색상 다양화 (트렌드 컬러)

❌ DON'T:
- Stanley/Yeti 직접 언급 (IP 위험)
- 가격 경쟁만으로 승부
- 품질 낮은 중국산 저가
```

### 키워드 전략

```
[메인 키워드]
- insulated tumbler
- stainless steel tumbler
- tumbler with handle

[롱테일 키워드]
- best tumbler for hot coffee
- tumbler that fits in cup holder
- tumbler for iced drinks all day
- cute tumbler for women
```

### 추천 공급처
- **CJ Dropshipping**: 다양한 옵션, 저가
- **Printify**: 커스텀 프린트 가능
- **US 로컬 공급자**: 품질 중시 시

### 💡 Pro Tip
> "40oz with handle" 키워드가 2024년 최고 인기
> Stanley 대안 포지셔닝이 효과적

## 2. 파티샵 (Party Supplies) 🎉

### 시장 특성

| 항목 | 내용 |
|------|------|
| **시장 규모** | 안정적 수요 |
| **경쟁 강도** | 🟡 중간 |
| **마진** | 40-60% |
| **시즌성** | 홀리데이 피크 (10-12월) |

### 성공 전략

```
✅ DO:
- 테마별 세분화 (생일, 베이비샤워, 웨딩)
- 파티 키트/세트 구성
- 개인화 옵션 (이름, 날짜)
- 인스타그래머블 디자인

❌ DON'T:
- 디즈니/캐릭터 무단 사용 (IP 위험)
- 단품 판매만 (객단가 낮음)
- 배송 지연 (파티는 날짜 고정)
```

### 테마별 인기 아이템

| 테마 | 인기 아이템 |
|------|------------|
| **Birthday** | 풍선 세트, 배너, 테이블웨어 |
| **Baby Shower** | 젠더리빌 세트, 게임 키트 |
| **Wedding** | 포토부스 소품, 테이블 넘버 |
| **Holiday** | 크리스마스 데코, 할로윈 소품 |

### 마케팅 전략

```
[Pinterest 필수]
- 파티 아이디어 핀 (트래픽 소스 1위)
- 무드보드 스타일 콘텐츠

[UGC 활용]
- 고객 파티 사진 리그램
- 리뷰 사진 요청 (할인 제공)

[시즌 준비]
- 홀리데이: 2개월 전 준비
- 졸업 시즌: 3-5월 집중
```

### 💡 Pro Tip
> 파티 날짜는 변경 불가 → **배송 시간 명확히**
> "Arrives by [Date]" 보장 못하면 판매 금지

## 3. 셀프케어 (Self-Care) 🧘‍♀️

### 시장 특성

| 항목 | 내용 |
|------|------|
| **시장 규모** | 급성장 중 |
| **경쟁 강도** | 🟡 중간 |
| **마진** | 50-70% |
| **시즌성** | 연중 (명절 선물 피크) |

### 성공 전략

```
✅ DO:
- 기프트 세트 구성 (높은 객단가)
- "Self-care Sunday" 브랜딩
- 인플루언서 협업
- 구독 박스 모델 고려

❌ DON'T:
- 의약품 효능 주장 (FDA 위반)
- 피부 효과 과장 광고
- 저품질 스킨케어 (리뷰 폭탄)
```

### 인기 상품 카테고리

| 카테고리 | 예시 | 마진 |
|----------|------|------|
| **Bath & Body** | 배스밤, 바디스크럽 | 60-70% |
| **Aromatherapy** | 캔들, 디퓨저 | 50-60% |
| **Relaxation** | 아이마스크, 마사지 도구 | 40-50% |
| **Skincare Tools** | 구아샤, 페이스롤러 | 50-60% |

### 키워드 전략

```
[감성 키워드]
- self care gift set
- relaxation gifts for women
- spa day at home
- stress relief gifts

[기능 키워드]
- jade roller benefits
- bath bomb gift set
- aromatherapy diffuser
```

### 주의사항

```
⚠️ 규제 주의:

1. 화장품 규정
   - "FDA approved" 허위 표기 금지
   - 성분 표기 의무

2. 효능 주장
   ❌ "Cures acne"
   ❌ "Anti-aging proven"
   ✅ "Helps relax tense muscles"
   ✅ "Luxurious spa experience"

3. 알레르기 경고
   - 성분 리스트 필수
   - 알레르기 주의 문구
```

### 💡 Pro Tip
> "Gift for her" 키워드 + 기프트 박스 패키징
> = 객단가 2배, 전환율 1.5배

## 4. 베이비 (Baby Products) 👶

### 시장 특성

| 항목 | 내용 |
|------|------|
| **시장 규모** | 안정적 대형 시장 |
| **경쟁 강도** | 🔴 높음 |
| **마진** | 30-50% |
| **시즌성** | 연중 (선물 시즌 피크) |

### 성공 전략

```
✅ DO:
- 안전 인증 강조 (CPSC, ASTM)
- 유기농/친환경 포지셔닝
- 출산선물 세트 구성
- 신뢰 구축 (리뷰 중요)

❌ DON'T:
- 안전 기준 미달 제품
- 미인증 장난감/젖병
- 과장 광고 ("최고의", "1위")
```

### 인기 상품 분석

| 상품 | 수요 | 마진 | 주의사항 |
|------|------|------|----------|
| **의류** | 높음 | 40% | 사이즈 차트 필수 |
| **장난감** | 높음 | 50% | CPSC 인증 확인 |
| **수유용품** | 중간 | 30% | FDA 규정 |
| **침구류** | 중간 | 45% | 안전 기준 |

### 안전 규정 체크리스트

```
[미국 베이비 제품 규정]

□ CPSC (Consumer Product Safety Commission)
  - 모든 어린이 제품 필수

□ CPSIA (Children's Product Safety Improvement Act)
  - 납, 프탈레이트 제한

□ ASTM F963 (장난감 안전 기준)
  - 장난감 필수 인증

□ Flammability Standards
  - 의류, 침구류 난연 기준
```

### 타겟 고객 세그먼트

```
1. First-time Parents
   - 정보 제공 콘텐츠 효과적
   - 체크리스트/가이드 제공

2. Gift Givers
   - 기프트 세트 구성
   - 기프트 박스 옵션

3. Eco-conscious Parents
   - 유기농/친환경 강조
   - 지속가능성 메시지
```

### 💡 Pro Tip
> 베이비 시장은 **신뢰가 전부**
> 리뷰 50개 이상 + 4.5점 이상이 전환의 핵심

## 6. 카테고리 공통 전략

### 가격 설정 공식

```
[권장 마크업]

저가 (< $10): 3-4배 마크업
중가 ($10-50): 2-3배 마크업  
고가 (> $50): 1.5-2배 마크업

예시:
원가 $5 → 판매가 $19.99 (4배)
원가 $20 → 판매가 $49.99 (2.5배)
원가 $50 → 판매가 $89.99 (1.8배)
```

## 7. 카테고리 선택 의사결정 트리

```
Q1. 경험이 있는 카테고리?
    ├─ 예 → 해당 카테고리 시작
    └─ 아니오 → Q2로

Q2. 투자 가능 예산?
    ├─ < $500 → 텀블러, 셀프케어 (저가 시작)
    └─ > $500 → Q3로

Q3. 콘텐츠 제작 가능?
    ├─ 예 → 주방용품, 파티샵 (비주얼 중요)
    └─ 아니오 → Q4로

Q4. 안전 규정 감수 가능?
    ├─ 예 → 베이비 (높은 진입장벽 = 낮은 경쟁)
    └─ 아니오 → 텀블러, 셀프케어
```
