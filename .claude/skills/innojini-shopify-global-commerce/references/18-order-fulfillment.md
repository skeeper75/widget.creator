# 발주 처리 가이드

*스파미 Shopify 오프라인 강의 기반 (2025년 12월)*

> 발주 = 3일 여유 있음
> 급하지 않게, 정확하게

---

## 1. 발주 기본 원칙

### 시간 여유

```
주문 들어오면:
❌ "빨리 발주해야지!"
❌ "퇴근하고 바로 해야지!"

✅ 3일까지 여유 있음
✅ 확인하고 정확하게

→ 미국 고객은 배송에 여유로움
→ 하루 이틀 늦어도 재촉 거의 없음
```

### 일일 루틴

```
[권장 루틴]
- 정해진 시간에 한 번
- 예: 아침 9시 또는 저녁 8시
- 몰아서 처리

[효율적 순서]
1. 신규 주문 확인
2. 사기 경고 체크
3. 주소 검증
4. 일괄 발주
5. 트래킹 업데이트
```

---

## 2. 주문 확인 체크리스트

### 발주 전 확인

```
□ 사기 경고 (빨간불/초록불)
□ 배송 주소 유효성
□ 주 코드 + 우편번호 일치
□ 상품 재고 확인
□ 가격 변동 확인
```

### 사기 경고 대응

```
[경고 없음]
→ 바로 발주

[초록불]
→ 대부분 OK, 주소만 확인

[빨간불]
→ 상세 확인 필요
→ 판단 후 발주 또는 캔슬
```

---

## 3. 재고 확인

### 소싱처 재고 체크

```
발주 시점에 재고 없을 수 있음!

[확인 순서]
1. 소싱처 접속
2. 상품 검색
3. 재고 상태 확인
4. 가격 변동 확인
```

### 재고 없을 때

```
[옵션 1] 다른 공급처 찾기
- 동일 상품 다른 셀러
- 가격 비교

[옵션 2] 고객 협상
- 품절 안내
- 부분 취소 제안
- 대체 색상 제안

[옵션 3] 캔슬
- 전체 환불
- 사유 안내 이메일
```

## 4. 구독구매 함정 주의 ⚠️

### One-time Purchase 필수

```
발주 화면에서:

❌ Subscribe & Save (선택 금지!)
✅ One-time Purchase (반드시!)

→ 실수로 구독 선택하면
→ 다음 달에도 자동 발주
→ 계속 돈 빠짐
```

### 쿠폰 함정

```
"15% OFF with Subscribe & Save"

→ 쿠폰 적용 = 자동 구독 동의
→ 쿠폰 조건 반드시 읽기
→ 일회성 쿠폰만 사용
```

### 구독 걸리기 쉬운 상품

```
재구매 상품군:
- 화장품
- 사료/펫용품
- 건강식품
- 생필품
- 청소용품

→ 이 카테고리 발주 시 특히 주의
```

## 5. 배송 지연 대응

### 7일 이상 지연 시

```
[프로세스]

1. 고객에게 이메일
   "배송이 예상보다 7일 지연됩니다.
    그래도 배송 진행할까요?"

2. 3일 대기
   - 응답 오면 그대로 진행
   - 응답 없으면 판단

3. 무응답 시
   - 캔슬 고려
   - 또는 발송 후 안내
```

### 지연 안내 이메일

```
Subject: Update on Your Order #[Number]

Hi [Name],

We wanted to let you know that your order 
may take about [X] days longer than expected.

New estimated delivery: [Date]

Would you like us to:
A) Proceed with the order
B) Cancel for a full refund

Please reply within 3 days.

Thank you for your patience!
```

## 6. 세트 상품 처리

### 세트 품절 시

```
[상황]
판매: 텀블러 2개 세트
소싱 시점: 2개 세트 $20
발주 시점: 1개만 $12

[해결]
$12 × 2 = $24
세트 가격 $20보다 비쌈

→ 마진 계산
→ 손해 아니면 1개×2 발주
→ 손해면 협상 또는 캔슬
```

### 구성 변경 안내

```
Subject: Your Order - Shipping Update

Hi [Name],

Your 2-piece set will ship as 
2 separate packages.

📦 Package 1: Arriving [Date 1]
📦 Package 2: Arriving [Date 2]

You were only charged once!

Thank you!
```

## 8. 트래킹 관리

### 트래킹 번호 입력

```
발주 후:
1. 공급처에서 트래킹 받음
2. Shopify 주문에 입력
3. 고객에게 자동 발송

[입력 위치]
Orders → [주문] → Fulfill → Add tracking
```

### 분리 배송 시

```
여러 패키지 = 여러 트래킹

1. Add tracking → 첫 번째 번호
2. Add another tracking → 두 번째
3. 각각 배송사 선택
4. 고객에게 모두 전송
```

## 9. 발주 실수 방지

### 흔한 실수

| 실수 | 결과 | 방지 |
|------|------|------|
| 주소 오타 | 배송 실패 | 복사-붙여넣기 |
| 구독 선택 | 매달 결제 | One-time 확인 |
| 수량 오류 | 과다/부족 | 이중 확인 |
| 옵션 오류 | 색상/사이즈 틀림 | 상세 대조 |

### 발주 전 최종 체크

```
□ 수령인 이름 정확
□ 주소 완전 (호수 포함)
□ 주 + 우편번호 일치
□ 상품/옵션 일치
□ 수량 정확
□ One-time Purchase
```

```

[자동 권장]
- 샵 5개+

- 루틴 완성 후
```

## 11. 발주 시간 관리

### 예상 소요 시간

```
주문 1건:
- 확인: 1분
- 발주: 2-3분
- 트래킹 입력: 1분
= 약 5분/건

일 10건 = 50분
일 20건 = 1시간 40분
```

### 효율화 팁

```
1. 같은 소싱처 묶어서 발주
2. 주소 복사-붙여넣기
3. 브라우저 자동완성 활용
4. 템플릿 이메일 준비
5. 정해진 시간에 몰아서
```
