## 1. AutoDS SPARK 프로그램

### SPARK란?
AutoDS의 프리미엄 교육 + 셋업 프로그램으로, 초보자가 빠르게 드랍쉬핑을 시작할 수 있도록 지원합니다.

### 비용 구조

| 항목 | 비용 | 설명 |
|------|------|------|
| **SPARK 초도비용** | **$2,500** | 일회성 가입비 |
| **월 구독료** | $19.90~$69.90 | 플랜별 차등 |
| **자동 주문 처리** | 추가 옵션 | 별도 과금 |

### SPARK 포함 내용
- ✅ 1:1 멘토링 세션
- ✅ 샵 초기 셋업 지원
- ✅ 위닝 제품 리스트 제공
- ✅ 마케팅 전략 가이드
- ✅ 커뮤니티 액세스

### SPARK 적합 대상
- 빠른 시작을 원하는 완전 초보자
- 시행착오 비용을 줄이고 싶은 분
- 멘토링/코칭이 필요한 분

### ⚠️ 주의사항
- $2,500은 환불 불가 (대부분의 경우)
- 성공을 보장하지 않음
- AutoDS 구독료는 별도

## 2. 아마존 기반 소싱의 장점

### 왜 아마존에서 소싱하는가?

| 장점 | 설명 |
|------|------|
| **신뢰도** | 아마존 Prime 배송 = 고객 신뢰 |
| **배송 속도** | 미국 내 2-5일 배송 가능 |
| **반품 처리** | 아마존 반품 시스템 활용 |
| **재고 안정성** | 품절 리스크 낮음 |
| **품질 보장** | 아마존 FBA 상품 품질 검증됨 |

### 아마존 소싱 전략

```
1. 아마존에서 제품 찾기
   └─ 가격 경쟁력 있는 제품 선별
   
2. Shopify에 마크업 가격으로 등록
   └─ 권장 마크업: 30-50%
   
3. 주문 발생 시 아마존에서 직접 구매
   └─ 고객 주소로 직배송 설정
   
4. 아마존 트래킹 번호를 Shopify에 등록
```

### 아마존 소싱 마진 계산 예시

```
아마존 판매가:      $25.00
Shopify 판매가:    $39.99
─────────────────────────
총 마진:           $14.99
- Shopify 수수료:  $1.46 (2.9% + $0.30)
- 기타 비용:       $1.00 (예상)
─────────────────────────
순 마진:           $12.53 (31.3%)
```

### ⚠️ 아마존 소싱 주의사항
- **아마존 TOS 위반 가능성** - 계정 정지 리스크
- **가격 변동** - 아마존 가격이 수시로 변경됨
- **재고 변동** - 갑작스러운 품절 가능
- **반품 처리** - 고객 반품 시 복잡해질 수 있음

## 3. 플랫폼별 소싱 전략 비교

### 초보자 추천 경로

```
[1단계] CJ Dropshipping (무료)
    ↓ 월 매출 $1,000 달성
[2단계] Zendrop Pro ($49/월)
    ↓ 월 매출 $5,000 달성
[3단계] AutoDS Advanced ($69.90/월)
    ↓ 월 매출 $10,000+ 
[4단계] 자체 공급망 구축
```

### 상품 유형별 추천 플랫폼

| 상품 유형 | 추천 플랫폼 | 이유 |
|----------|------------|------|
| **일반 소비재** | CJ Dropshipping | 무료 + 다양한 제품 |
| **커스텀/POD** | Printify | POD 전문 |
| **US 빠른 배송** | Zendrop | US 창고 보유 |

| **대량 자동화** | AutoDS | 자동화 기능 강력 |

## 4. AutoDS 상세 기능

### 플랜별 기능 비교 (Shopify 기준)

| 기능 | Import ($19.90) | Starter ($29.90) | Advanced ($69.90) |
|------|-----------------|------------------|-------------------|
| 제품 수 | 200개 | 400개 | 1,000개 |
| 스토어 수 | 1개 | 1개 | 다중 |
| 자동 가격/재고 | ✅ | ✅ | ✅ |
| AI 제목/설명 | ✅ | ✅ | ✅ |
| 자동 주문 처리 | ❌ | 옵션 | ✅ |
| VIP 지원 | ❌ | ❌ | ✅ |

### AutoDS 핵심 기능

1. **자동 가격 모니터링**
   - 공급처 가격 변동 시 자동 업데이트
   - 마진율 기준 자동 조정 가능

2. **자동 재고 동기화**
   - 품절 상품 자동 비활성화
   - 재입고 시 자동 활성화

3. **AI 제품 최적화**
   - 제목/설명 자동 생성
   - SEO 최적화 적용

4. **자동 주문 처리** (유료 애드온)
   - 주문 발생 → 자동 발주
   - 트래킹 번호 자동 입력

### AutoDS 지원 공급처
- AliExpress
- Amazon (US/UK/DE 등)
- Walmart
- Home Depot
- Costco
- 기타 50+ 공급처
