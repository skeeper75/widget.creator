# GSK 정산 시스템 가이드

*스파미 Shopify 오프라인 강의 기반 (2025년 4월)*

> ⚠️ **주의**: GSK는 미국 캘리포니아 법인으로, Shopify Payments 인증을 대행합니다.
> 법무법인 동인의 법률 검토 의견서를 통해 적법성이 확인되었습니다.

---

## 1. GSK란?

GSK(Global Seller Korea)는 한국인이 미국 시장에서 Shopify 드랍쉬핑을 할 때 **Shopify Payments 인증을 대행**해주는 서비스입니다.

### 왜 GSK가 필요한가?
- Shopify Payments는 미국 사업자/SSN 필요
- 한국인은 직접 Shopify Payments 개설 불가
- GSK를 통해 미국 법인 명의로 결제 시스템 사용

---

## 2. 정산 구조 (후불 방식)

GSK는 **후불 정산 방식**을 사용합니다. 매출이 발생한 후 정산받는 구조입니다.

### 수수료 체계

| 항목 | 비용 | 설명 |
|------|------|------|
| **GSK 수수료 (첫 샵)** | $150/월 | 첫 번째 샵 고정 월비용 |
| **GSK 수수료 (추가 샵)** | 매출의 5% | 두 번째 샵부터 적용 |
| **세금** | 매출의 7% | 미국 판매세 (Sales Tax) |
| **회계사 비용** | 매출의 3~5% | 매출 구간별 차등 적용 |
| **송금 수수료** | $50/회 | 한국 계좌로 송금 시 |

### 매출 구간별 회계사 비용

| 월 매출 구간 | 회계사 비용 |
|-------------|------------|
| $0 ~ $10,000 | 5% |
| $10,000 ~ $50,000 | 4% |
| $50,000+ | 3% |

## 3. 실수령액 계산 예시

### 예시 1: 뷰티샵 (월 매출 $5,000)
```
총 매출:           $5,000
- 카드 수수료(2.9%): $145
- GSK 수수료:       $150 (첫 샵 고정)
- 세금(7%):         $350
- 회계사(5%):       $250
- 송금 수수료:      $50
─────────────────────────
실수령액:          $4,055 (81.1%)
```

### 예시 2: 베이비샵 (월 매출 $20,000)
```
총 매출:           $20,000
- 카드 수수료(2.9%): $580
- GSK 수수료(5%):   $1,000 (두 번째 샵)
- 세금(7%):         $1,400
- 회계사(4%):       $800
- 송금 수수료:      $50
─────────────────────────
실수령액:          $16,170 (80.9%)
```

### 예시 3: 캠핑샵 (월 매출 $50,000)
```
총 매출:           $50,000
- 카드 수수료(2.9%): $1,450
- GSK 수수료(5%):   $2,500
- 세금(7%):         $3,500
- 회계사(3%):       $1,500
- 송금 수수료:      $50
─────────────────────────
실수령액:          $41,000 (82%)
```

## 4. 정산 일정

| 단계 | 일정 | 설명 |
|------|------|------|
| 매출 발생 | D+0 | Shopify Payments로 결제 |
| 정산 준비 | D+3~7 | GSK에서 정산 내역 정리 |
| 세금 공제 | - | 자동 계산 |
| 송금 | D+7~14 | 한국 계좌로 입금 |

---

## 5. 주의사항

### ✅ 장점
- 미국 Shopify Payments 직접 개설 불필요
- 법적 검토 완료 (법무법인 동인)
- 세금/회계 처리 대행
- 환율 변동 리스크 최소화

### ⚠️ 주의점
- **첫 샵 $150 고정비** - 매출이 적으면 비율 부담 증가
- **추가 샵 5%** - 다수 샵 운영 시 수수료 누적
- **송금 수수료 $50** - 월 1회 이상 송금 시 추가 발생
- **환율** - 정산 시점 환율 적용

### 🚫 부적합 케이스
- 월 매출 $1,000 미만: 수수료 비율 과다
- 다수 저매출 샵 운영: 5% 수수료 누적
- 높은 마진 유지 필요 시: 총 공제 약 18~20%

## 6. GSK vs 직접 미국 법인 설립

| 항목 | GSK 사용 | 미국 법인 직접 설립 |
|------|----------|-------------------|
| 초기 비용 | 낮음 | 높음 ($2,000+) |
| 월 비용 | $150 + 5% | 회계사비 + 세금만 |
| 관리 부담 | 낮음 | 높음 |
| 확장성 | 제한적 | 유연 |
| 적합 대상 | 초보/테스트 | 월 $10만+ 규모 |
