# 지식재산권 대응 가이드

*스파미 Shopify 오프라인 강의 - 변리사 특강 기반 (2025년 12월)*

> 소송 걱정보다 신고 대응이 현실
> 3일 내 응답 + 제품 삭제 = 해결

---

## 1. 지식재산권 3가지 유형

### Shopify 신고 근거

```
Shopify IP 신고 시스템:
1. 상표권 (Trademark)
2. 트레이드 드레스 (Trade Dress)
3. 저작권 (Copyright)

→ 이 3가지로만 신고 가능
→ 다른 쇼핑몰도 비슷
```

### 1.1 상표권 (Trademark)

```
정의: 내 제품과 남의 제품을 구분하는 표시

예시:
- Nike 로고
- Disney 문자
- Apple 사과 마크

표시:
- ™ = 미등록 상표
- ® = 등록 상표
```

### 1.2 트레이드 드레스 (Trade Dress)

```
정의: 제품의 전체적인 특징/디자인

예시:
- 샤넬 가방 디자인 (로고 없어도 알아봄)
- 루이비통 패턴
- 코카콜라 병 모양

→ 상표 없이 디자인만 베끼는 경우 신고용
```

### 1.3 저작권 (Copyright)

```
정의: 창작물에 대한 권리

예시:
- 제품 사진
- 상세페이지 이미지
- 상품 설명 글
- 사용법 인포그래픽

주의: 짧은 문구는 저작권 인정 안 될 수 있음
```

---

## 2. 드랍쉬핑에서의 리스크

### 상표권/트레이드 드레스 = 낮은 위험

```
드랍쉬핑 특성:
- 공급자에게서 상품 가져옴
- 그 상품 그대로 판매
- 고객이 받는 건 동일 상품

→ 상표권 침해 여지 없음
→ 공급자에게 사든 나한테 사든 같은 물건
```

### 예외: 짝퉁 상품

```
⚠️ 문제되는 경우:

내가 가져온 상품이 알고 보니 짝퉁
→ 공급자도 문제
→ 나도 문제 (광고하고 팔았으니까)

→ 소싱처 신뢰도 확인 필요
```

### 저작권 = 주의 필요

```
드랍쉬핑에서 저작권 문제:

1. 공급자 이미지 무단 사용
   - 공들여 찍은 제품 사진
   - 예쁘게 연출한 이미지
   - 상세한 사용법 인포그래픽

2. 공급자가 문제 삼을 수 있음
   - "내가 만든 이미지인데 왜 너가 써?"
   - 기분 나쁘면 신고 가능
```

## 3. 소송 현실

### 소규모 판매자 소송 = 비현실적

```
소송의 경제학:
- 이기면 손해배상 받음
- 손해배상액 = 침해 규모에 비례

소규모 판매자:
- 판매량 적음 = 손해배상액 적음
- 소송 비용 > 받을 금액
- 권리자 입장에서 실익 없음

→ 이론적으로 가능하지만 현실에서 안 일어남
```

### 대기업이 소송하는 방식

```
2000년대 오픈마켓 시대:

권리자들 전략:
- 개별 판매자 소송 X (건덕지 없음)
- 플랫폼 (아마존, G마켓) 소송 O

"너네 플랫폼에서 짝퉁 팔리는데 
 너네도 책임 있다"

→ 플랫폼 = 큰 회사 = 돈 있음
→ 소송 실익 있음
```

### 판례 결과

```
현재 정리된 법리:

플랫폼 책임 조건:
1. 침해 행위를 알게 됨
2. 중단할 기술적 방법 있음
3. 그럼에도 조치 안 함

→ 플랫폼들이 "신고 시스템" 만듦
→ 자기네 면책용
→ 이게 지금의 IP 신고 절차
```

## 4. 신고 프로세스

### Shopify IP 신고 절차

```
[권리자/대리인]
    ↓ 온라인 신고
[Shopify]
    ↓ 확인 (권리자 맞는지, 실제 침해인지)
[상품 삭제]
    ↓ + 판매자에게 통지
[소명 요청]
    ↓ 3일 내 응답 요구
[판매자 대응]
```

### 3일 규칙 ⚠️

```
신고 접수 → 3일 내 응답 요구

응답 안 하면:
- 상품 삭제 확정
- 반복 시 계정 위험
```

### 쿠팡 사례 (변리사 실제 경험)

```
드랍쉬핑 하던 분:
- 제품 많아서 이메일 놓침
- 신고 왔는데 모름
- 제품 하나 삭제됨 (몰랐음)

또 신고 옴 → 또 놓침

2번 놓치니까 → 계정 폐쇄!

+ 연동된 개인정보로 재가입도 차단
```

### 2 Strike Out

```
쿠팡: 2번 무시 → 계정 폐쇄

Shopify: 
- 정확한 횟수 미공개
- 약관에 "계정 폐쇄 가능" 명시
- 비슷한 조치 예상
```

## 5. 현실적 대응 전략

### 가장 효율적인 대응

```
신고 왔을 때:

❌ 법리적으로 다투기
   - 영어로 구구절절 설명
   - 받아들여질 보장 없음
   - 시간/에너지 낭비

✅ 인정 + 삭제
   - "몰랐습니다"
   - "삭제하겠습니다"
   - 제품 하나 포기
   - 사업 계속
```

### 삭제하면 끝?

```
Q: 삭제하면 그걸로 끝?
A: 대부분 YES

이유:
- 권리자 목적 = 판매 중단
- 삭제되면 목적 달성
- 소송까지 갈 이유 없음
- 대리인도 일일이 추적 안 함
```

### 눈에 찍히지 말 것

```
반복 무시 → 악질로 찍힘

한 번 삭제 요청 받으면:
- 바로 삭제
- "삭제했다" 답변까지 하면 더 좋음
- 그리고 잊어버림

→ 다음 제품 판매에 집중
```

## 6. 신고 대행업체 이해

### 신고 = 대량으로 뿌려짐

```
대기업이 직접 신고? X

실제:
- AI로 침해 의심 자동 검색
- 엑셀에 100개씩 링크 정리
- 대행업체에 "여기 다 신고해줘"
- 월 수백 건 자동 처리

→ 대기업이 "나"를 알고 신고한 게 아님
→ 자동화된 시스템
→ 겁먹을 필요 없음
```

### 변리사 실제 경험

```
"대리인 통해서 요청받으면
 엑셀에 100개 웹페이지 주고
 '여기 다 신고해줘' 하는 경우 많았어요"
```

## 7. 저작권 실전 케이스

### 중고 명품샵 사례

```
[문제]
중고 명품 판매 사이트들
- 공식 홈페이지 사진 무단 사용
- 명품 브랜드에서 문제 삼음

[해결]
- 직접 제품 촬영으로 전환
- 자체 이미지 사용
- 문제 해결
```

### AI 변형해도 괜찮을까?

```
Q: AI로 이미지 변형하면?
A: 결과물만 봄

- AI로 했든 포토샵이든 상관없음
- 최종 결과물이 비슷하면 문제
- 입장 바꿔 생각해보기:
  "내 이미지를 누가 이렇게 변형해서 쓰면 짜증날까?"
  → 짜증나면 문제될 가능성 있음
```

## 8. 화이트라벨 전략

### 화이트라벨이란?

```
공급자 브랜드 없이 → 내 브랜드로 판매

장점:
- 상표권 문제 없음 (내 브랜드니까)
- 가격 비교 어려움
- 브랜딩 가능
```

### IP 리스크 최소화

```
[화이트라벨 체크리스트]

□ 공급자 로고 없는 제품 선택
□ 자체 브랜드명 사용
□ 자체 촬영 이미지 사용
□ 자체 작성 상품설명
```

## 9. 초상권

```
수강생 + 100개 샵 운영 경험:
- 초상권 신고/요청 = 0건
- 아직 문제 없었음
```

### 예방책

```
초상권 걱정되면:
- 사람 안 나오는 카테고리 선택
- 사무용품
- 가든 용품
- 주방용품
- 공구류
```

## 11. 신고 받았을 때 즉시 행동

### 1시간 내 해야 할 일

```
1. 이메일 전체 읽기
   - 어떤 상품인지
   - 누가 신고했는지
   - 무엇을 요구하는지

2. 해당 상품 비활성화
   - 삭제 전 일단 숨김
   - 증거 보존

3. 응답 초안 작성
   - "삭제하겠다" 또는
   - "드랍쉬퍼입니다" 설명
```

### 드랍쉬퍼 설명 (필요시)

```
권리자가 궁금해하는 것:
"너네한테 공급 안 했는데, 이거 어디서 났어?"

답변:
"저는 드랍쉬퍼입니다.
 판매되면 귀사에 발주 넣으려 했습니다.
 문제가 된다면 바로 삭제하겠습니다."

→ 드랍쉬핑 모델 설명
→ 불법 루트 아님 증명
→ 협조적 태도 표시
```
