# 미국 주소 체계 심화 가이드

*스파미 Shopify 오프라인 강의 기반 (2025년 12월)*

> 주소 오류 = 배송 실패 = 차지백 위험
> 주소 체계 이해가 발주의 기본

---

## 1. 기본 주소 구조

### 예시 분석

```
3442 Caraway Ln #201, Yorba Linda, CA 92886, United States

분해:
- 3442 = 건물 번호
- Caraway Ln = 도로명 (Caraway 길)
- #201 = 호수 (Suite/Unit/Shop)
- Yorba Linda = 동네 이름 (절대 줄여 쓰지 않음)
- CA = 주 약자 (California)
- 92886 = 우편번호 (ZIP Code)
```

### 호수가 없는 경우

```
❌ 한국 사고방식:
"호수가 없네? 이상하다, 물어봐야지"

✅ 미국 현실:
단독 주택이 많아서 호수 없는 게 정상
건물 번호만 있으면 OK
```

## 2. LA 주의! 🔴 (초보 실수 1위)

### 치명적 착각

```
주 코드 "LA"를 보고:
❌ "아, 로스앤젤레스구나" 
✅ "LA = Louisiana (루이지애나)"

로스앤젤레스는 CA (California)!
```

### 완전히 다른 곳

```
LA (Louisiana) = 미국 남부, 뉴올리언스
CA (California) = 미국 서부, 로스앤젤레스

→ 잘못 보내면 반대편으로 배송
→ 재발송 비용 + 시간 손실
```

### 대응 방법

- 주 코드 암기는 불가능 (50개)
- **우편번호와 주 코드 일치 확인**이 핵심
- CA는 9로 시작, LA는 7로 시작

## 3. 우편번호(ZIP) 규칙

### 주별 우편번호 시작 숫자

| 시작 숫자 | 대표 주 |
|-----------|---------|
| 0 | 동부 (MA, CT, NJ 등) |
| 1 | 동부 (NY, PA) |
| 2 | 동부 (DC, VA, NC) |
| 3 | 남부 (FL, GA, AL) |
| 4 | 중부 (MI, OH, IN) |
| 5 | 중부 (MN, WI, IA) |
| 6 | 중부 (IL, MO, KS) |
| 7 | 남부 (TX, LA, OK) |
| 8 | 서부 (CO, AZ, NV) |
| 9 | 서부 (CA, WA, OR) |

### 빠른 검증

```
예: CA 12345
→ CA는 9로 시작해야 함
→ 1로 시작? = 주소 오류!
```

### 80% 커버 규칙

> "주 코드 + 우편번호 첫 자리만 맞으면 80% 해결"

## 4. 건물 번호 규칙

### 홀수/짝수 규칙

```
도로 기준:
- 왼쪽 = 짝수 (6030, 6032, 6034...)
- 오른쪽 = 홀수 (6031, 6033, 6035...)
- +2씩 증가
```

### 건물 번호 점프

```
예: 6036 다음이 6042?

→ 중간에 큰 집이 있어서 2칸 차지
→ 6038, 6040 건물 자리를 큰 집이 먹음
→ 오류 아님!
```

### 뉴욕 도심지 특수 케이스

```
뉴욕 같은 도심지:
- 한 자리에 건물 2개
- 3011-1, 3011-2 형태

주의:
- 일부 플랫폼(아마존)은 하이픈 인식 안 함
- 3011 1, 3011 2 (띄어쓰기)로 표시되기도 함
- 3011의 1호가 아니라 3011-1 건물!
```

## 5. ChatGPT 활용법

### 주소 분석

```

"이 주소 체계를 분석해줘:
3442 Caraway Ln #201, Yorba Linda, CA 92886"

→ 건물번호, 도로명, 호수, 동네, 주, 우편번호
   모두 분해해서 설명해줌
```

### 구글맵 빈 공간 확인

```
[상황]
구글맵에서 주소 검색했는데 빈 공간?

❌ "여기 아무것도 없네, 캔슬해야지"

✅ ChatGPT에게:
"이 주소로 주문이 들어왔는데 
구글맵에 빈 공간이야. 배송 보내도 돼?"

→ "해당 위치에 2023년 신축 타운하우스 완공,
   최근 $XXX에 매매됨" 정보 제공

→ 신축이라 구글맵 미업데이트였던 것!
```

### 주소 검증 프롬프트

```
"다음 주소가 유효한지 확인해줘:
[주소 붙여넣기]

- 주 코드와 우편번호 일치하는지
- 실제 존재하는 주소인지
- 배송 가능한 곳인지"
```

## 6. 푸에르토리코 (PR)

### 기본 정보

```
- 미국령 (Territory)
- 주 코드: PR
- 우편번호: 00XXX 형식
- 미국 내 배송 가능!
```

### 특수성

```
언어: 스페인어 기반
→ 주소가 영어로 안 읽힘
→ ChatGPT로 분석 필수

예:
"Calle Luna, San Juan, PR 00901"
→ Calle = Street (스페인어)
```

### 캔슬하지 마세요!

```
실제 사례:
- 마리아라는 고객이 PR에서 주문
- $319 상당
- 주소 이상해 보여도 정상 배송됨

→ 심심치 않게 주문 들어옴
→ 지도에 섬으로 보여도 미국 배송!
```

### PR 발주 방법

```
1. 주소 전체를 ChatGPT에 붙여넣기
2. "발주할 수 있게 주소 체계 분석해줘"
3. 각 요소 확인 후 정상 발주
```

## 7. 라커(Locker) 배송

### 라커란?

```
- Amazon Locker
- UPS Access Point
- 건물 내 무인 보관함

→ 분실/도난 방지용
→ 비밀번호로 열어서 수령
```

### 비밀번호 문제

```
중요!
- 비밀번호가 배송 도착 당일 아침에 이메일로 옴
- 그 이메일을 고객에게 전달해야 함
- 전달 안 하면 → 고객이 못 꺼냄
```

### 3일 규칙

```
라커에서 3일 내 미수령 시:
→ 자동 반송 (Return)
→ 캔슬 처리해야 함
→ 마진 손실
```

### 라커 배송 프로세스

```
1. 주문 시 라커 주소 확인
2. 발송
3. 배송 당일 비밀번호 이메일 확인
4. 고객에게 즉시 전달
5. 3일 내 수령 확인
```

## 8. 군사 주소 (APO/FPO/DPO)

### 형식

```
CPT John Smith
Unit 12345 Box 6789
APO AE 09012

- APO: Army Post Office
- FPO: Fleet Post Office
- DPO: Diplomatic Post Office
- AE: Armed Forces Europe
- AP: Armed Forces Pacific
- AA: Armed Forces Americas
```

### 주의사항

| 항목 | 내용 |
|------|------|
| **배송사** | USPS만 가능 |
| **배송 시간** | 2-4주 |
| **트래킹** | 해외 구간 제한적 |
| **제한 품목** | 배터리, 위험물 불가 |

## 9. 문제 주소 유형

### 주소가 길로 끝남

```
예: "3442 Caraway Ln"
→ 건물 번호 없이 길 이름만

❌ 그냥 보내기
✅ 이메일로 확인 요청
```

### 창고/포워딩 주소

```
위험 신호:
- "Suite 1234" (4자리 이상)
- "c/o [회사명]"
- "Attn: [숫자]"
- MyUS, Shipito, Planet Express 등

→ 해외 재발송 목적
→ 차지백 위험 높음
→ 거부 또는 확인 필요
```

### 주소 확인 이메일

```
Subject: Please Confirm Your Shipping Address

Hi [Name],

Before we ship your order, we noticed 
your address may need verification:

[주소 표시]

Could you please confirm this is correct?
If any changes are needed, reply within 24 hours.

Thank you!
```

## 10. 반품 지정 장소

### 미국 반품 시스템

```
한국: 택배 기사가 문 앞에서 수거
미국: 고객이 직접 지정 장소로 가져감

→ 월마트, UPS Store 등
→ QR 코드 스캔
→ 영수증 사진 촬영
```

### 반품률이 낮은 이유

```
반품하려면:
1. 차에 시동 걸기
2. 월마트까지 운전
3. 포장된 물건 들고 가기
4. QR 코드 스캔
5. 영수증 사진 찍기

→ 너무 귀찮음
→ 한국 대비 반품률 훨씬 낮음
```

### 반품 요청 시 증거 확보

```
[고객에게 요청]

1. 월마트 지점에 갖다 놓은 사진
2. QR 코드 스캔 완료 사진
3. 영수증 사진

→ 중간 분실 시 우리 손해 방지
→ 갖다 놓은 증거 필수!
```
