# 차지백 방어 완전 가이드

*스파미 Shopify 오프라인 강의 기반 (2025년 12월)*

> ⚠️ 차지백 = 가장 큰 리스크. 예방이 최선, 소명이 차선

---

## 1. 차지백이란?

고객이 카드사에 **"나 사기당했다"** 신고하여 강제 환불받는 제도

### 원래 목적
- 고객 보호 (미배송, 사기 피해)
- 카드사가 중재

### 악용 유형 (Friendly Fraud)
- 물건 받고 "안 받았다" 주장
- 가족이 몰래 구매 → "나 아니다"
- 반품 귀찮아서 차지백

---

## 2. 플로리다 특별 경고 🔴

### 왜 플로리다인가?

```
지리적 특성:
- 쿠바, 도미니카, 브라질과 가까움
- 해상/항공 운반 용이
- 물류 창고 밀집 지역

→ 물건 받아서 해외로 빼돌리기 최적
→ 찾을 수 없음 = 완전 범죄
```

### 실제 사례 (강의 중 언급)

> 플레이스테이션 $800짜리 판매
> → 플로리다에서 대량 주문
> → 사기꾼 커뮤니티에서 "여기 호구 있다" 소문
> → 동네 사기꾼들이 몰려옴

### 플로리다 주문 판단 기준

| 배송지 | 위험도 | 판단 |
|--------|--------|------|
| **가정집** | 🟢 낮음 | 추적 가능, 보통 OK |
| **창고/웨어하우스** | 🔴 높음 | 매우 주의 |
| **물류회사** | 🔴 높음 | 로직스/포워딩 = 위험 |
| **꼬다리 지역** | 🔴 높음 | 항구 근처 = 빼돌리기 루트 |

### 플로리다 가정집도 안심?
- 가정집은 **상대적으로** 안전
- 하지만 고가 상품은 여전히 주의

## 3. 사기 위험 신호 상세

### Shopify 자동 경고

```
주문 목록에서:

🟢 초록색 = 좀 아리까리함, 대부분 OK
🔴 빨간색 = 확실히 위험 신호
```

### 경고 뜨는 기준

| 신호 | 설명 |
|------|------|
| **주문/배송 위치 차이** | 뉴욕에서 주문 → LA로 배송 |
| **카드 여러 장 시도** | 3-4개 카드 시도 후 결제 성공 |
| **새 계정 + 고가** | 첫 주문이 $500+ |
| **배송 급행 요청** | "빨리 보내줘" 압박 |
| **VPN 사용** | IP 우회 접속 |

### 카드 시도 횟수가 중요한 이유

```
사기꾼 행동 패턴:
1. 주운 카드로 시도 → 실패
2. 다른 카드 시도 → 실패  
3. 또 다른 카드 → 성공!
4. 그 카드로 구매

→ "결제 시도 3회 이상" = 강력한 사기 신호
```

## 4. 저가 상품 = 안전지대

### 왜 저가가 안전한가?

> 28불짜리 국수 거름망에 사기 경고 떴지만...
> 사기꾼 입장: "이거 되팔아봤자 생계 유지 안 됨"

### 가격대별 리스크

| 가격대 | 사기 위험 | 이유 |
|--------|----------|------|
| **$30 이하** | 🟢 낮음 | 되팔기 타산 안 맞음 |
| **$30-80** | 🟢 낮음 | 권장 가격대 |
| **$100-200** | 🟡 중간 | 주의 필요 |
| **$300+** | 🔴 높음 | 초보 셀러 타깃 |
| **$500+** | 🔴🔴 매우 높음 | 고위험 |

### 초보 셀러가 고가 피해야 하는 이유
1. 홈페이지가 "만만해" 보임
2. 대응 미숙 → 쉬운 타깃
3. 소명 경험 없음

## 5. 홈페이지 신뢰도 = 최고의 방어

### 핵심 인사이트

```
사기꾼 심리:
"이 샵 허접해 보이네 → 보내고도 모르겠지"

소비자 심리:
"이 샵 허접해 보이네 → 안 살래"

→ 허접한 샵 = 손님은 안 오고 사기꾼만 옴
```

### 신뢰도 높이는 요소

| 요소 | 효과 |
|------|------|
| **완성된 About 페이지** | 실제 비즈니스 증명 |
| **Contact 정보** | 미국 전화번호, 이메일 |
| **정책 페이지** | 환불, 배송, 개인정보 |
| **상품 설명 상세** | 전문성 어필 |
| **블로그 포스팅** | 활성화된 샵 증명 |
| **리뷰** | 사회적 증거 |

### 신뢰도 = 사기 방어

```
잘 갖춰진 샵:
- 사기꾼: "여긴 들킬 것 같아"
- 소비자: "여긴 믿을 만해"

→ 손님은 오고 사기꾼은 피함
```

## 6. 차지백 발생 시 소명 전략

### 소명 필수 증거 2가지

```
1️⃣ 배송 증빙
   - 송장 번호 조회 결과
   - 배송 완료 사진 (있으면 확정 승소)
   - 배송사 공식 기록

2️⃣ 고객 소통 이력
   - 배송 전 문의 없었음 캡처
   - "물건 안 왔어" 없이 바로 차지백 = 의도적
   - 이메일 기록 전체
```

### 소명 논리

```
[소명서 핵심 주장]

"고객이 물건을 못 받았다면
정상적으로 저에게 먼저 연락했을 것입니다.
그러나 어떠한 문의도 없이 바로 차지백을 신청했습니다.
이는 의도적인 사기입니다."

+ 배송 완료 증빙 첨부
```

### 72일 대기 현실

```
항소 제출 후:
- Shopify: "72일 내 답변하겠음"
- 72시간 아님, 72일!
- 그동안 정산금 보류됨
- 이미 받은 정산은 마이너스 처리
```

## 7. 1% 임계치 규칙

### 차지백 비율 관리

```
전체 주문 중 차지백 비율:

1% 미만 → 정상 운영
1% 이상 → 노출 제한 위험
반복 시 → 샵 정지 가능
```

### 비율 계산

```
월 주문 100건 기준:
- 차지백 0건 = 0% ✅
- 차지백 1건 = 1% ⚠️ 경계
- 차지백 2건 = 2% 🔴 위험
```

### 비율 관리 전략
1. 고위험 주문 선제 캔슬
2. 사기 신호 있으면 확인 후 발송
3. 차지백 발생 시 즉시 항소

## 8. 패소 비용

### 차지백 패소 시 손실

```
[비용 구조]

1. 상품 원가: $XX (이미 발송)
2. 배송비: $X (이미 지출)  
3. 결제 수수료: 2.9% (환불 안 됨)
4. 분쟁 수수료: $15 (패소 시)
5. 환불금: 전액

→ 원가 + 배송비 + 수수료 + $15 = 총 손실
```

### 사기꾼도 패소하면 손해

```
사기꾼 패소 시:
- 물건 못 받음 (어차피 없음)
- $15 분쟁 수수료 부담

→ 28불짜리 차지백 시도:
  성공해도 28불, 실패하면 -15불
  = 타산 안 맞음
```

## 9. 빨간불 ≠ 100% 사기

### 실제 사례

> 빨간 경고 뜬 주문 → 고민 끝에 발송
> → 고객이 별점 5점 리뷰 남김
> → 그냥 경고 시스템의 과민 반응이었음

### 판단 기준

```
경고 떴을 때 체크:

□ 가격대가 낮은가? ($50 이하)
□ 배송지가 가정집인가?
□ 카드 시도 1회인가?
□ 플로리다 아닌가?

→ 대부분 체크되면 발송 OK
→ 다 안되면 캔슬 고려
```

### 캔슬 시 처리

```
[캔슬 이메일]

"재고가 없어서 취소되었습니다"

❌ "사기 의심돼서 취소함" (절대 금지)
```

## 10. 차지백 예방 체크리스트

### 발주 전

```
□ Shopify 경고 확인
□ 배송 주소 구글맵 확인
□ 플로리다 + 창고 = 추가 확인
□ 고가($200+) = 신중 판단
□ 카드 시도 횟수 확인
```

### 발주 후

```
□ 배송 시작 이메일 발송
□ 트래킹 번호 공유
□ 배송 완료 확인
□ 완료 사진 저장 (있으면)
```

### 문제 발생 시

```
□ 즉시 소명 준비
□ 배송 증빙 수집
□ 고객 소통 이력 캡처
□ 72일 대기 각오
□ 정산 마이너스 대비
```

## 11. 문제 샵 처리

### 차지백 누적된 샵

```
상황: 사기꾼들 사이에 "호구 샵" 소문 남
→ 정상 고객보다 사기꾼이 더 많이 옴
→ 비율 관리 불가능
```

### 해결책

```
1. 해당 샵 폐쇄 (문 닫기)
2. 새 샵으로 갈아타기
3. 예비 샵 미리 준비해두면 빠름
```

### 예비 샵 전략

```
[권장]
- 운영 샵 외에 2-3개 예비 생성
- 락(비밀번호) 걸어두기
- 플랜 최소 비용만 유지
- 문제 시 즉시 전환
```
