## 3. 차지백(Chargeback) 완전 가이드

### 기본 정보
| 항목 | 내용 |
|------|------|
| **Shopify 차지백 수수료** | **$15 USD** (미국 기준) |
| **환불 여부** | 승소 시 $15 환불 |
| **Fraud Protect** | 적격 Shop Pay 주문에 대해 차지백 수수료 환불 |
| **업계 평균 수수료** | $5-$50 |

### 차지백 비율 임계치
| 비율 | 상태 | 조치 |
|------|------|------|
| **< 0.65%** | 정상 | 모니터링 없음 |
| **≥ 0.65%** | 주의 | Visa VDMP 프로그램 진입 위험 |
| **≥ 1%** | 위험 | Shopify Payments 제한/해지 가능 |

### Visa VDMP (Dispute Monitoring Program)
| 기간 | 페널티 |
|------|--------|
| 1-4개월 | 페널티 없음 |
| 5-11개월 | 차지백당 $50 벌금 |
| 12개월+ | 심층 검토, Visa 결제 자격 박탈 가능 |

### 차지백 원인별 대응
| 원인 | 비율 | 대응 방법 |
|------|------|----------|
| **Friendly Fraud** | 23% | 배송 증빙, 고객 서명, 대화 기록 보관 |
| **Third-party Fraud** | 23% | 3D Secure, AVS 검증, 사기 분석 도구 |
| **Product Not Received** | - | 추적 번호 확인, 배송 증빙 |
| **Unrecognized** | - | 청구서 설명 명확화, 고객 연락 |

### 승률 통계
- **업계 평균 승률**: 약 30%
- **Friendly Fraud 승률**: 약 43.82%
- **Chargeflow 사용 시**: 75%+ 목표

## 4. Reddit & 커뮤니티 정보

### 주요 드랍쉬핑 Subreddit
| Subreddit | 설명 | 활용도 |
|-----------|------|--------|
| **r/dropship** | 드랍쉬핑 전문 커뮤니티 | 전략, 문제해결, 경험 공유 |
| **r/Entrepreneur** | 창업자 커뮤니티 | 비즈니스 전략, 네트워킹 |
| **r/shopify** | Shopify 전용 | 기술 문제, 앱 추천 |
| **r/ecommerce** | 이커머스 전반 | 트렌드, 마케팅 |
| **r/DropShipping101** | 초보자 커뮤니티 | 입문 가이드, Q&A |

### 커뮤니티에서 자주 언급되는 팁
1. **니치 선택**: 감정적 연결이 있는 분야 (반려동물, 아기용품) 추천
2. **마진**: 고가 제품 30-50% 마진 가능, 저가 제품은 경쟁 치열
3. **실패율**: 대부분의 드랍쉬핑 비즈니스가 실패 (인내심 필요)
4. **성공 시간**: 평균 3-6개월 후 수익 발생
5. **현실적 수익**: 초보자 월 $1,000-$2,000, 경험자 $2,000-$5,000+

### Reddit 수익 공유 사례
- High-ticket 드랍쉬핑: 월 $80,000-$120,000 매출, 50% 마진 (제품가 $1,000-$2,000)
- 일반 드랍쉬핑: 초보자 일 $20-$100 수익, 성공자 일 $2,000-$5,000
