# 현지화 체크리스트

*스파미 Shopify 오프라인 강의 기반 (2025년 12월)*

> 현지화 = 미국인처럼 보이기
> 신뢰 = 전환의 기본

---

## 1. 현지화란?

### 캄보디아 비유

```
캄보디아 사람이 한국에서 물건 판매:
- 주소: 캄보디아
- 연락처: 캄보디아
- 이메일: 캄보디아

→ 한국인이 신뢰할까?
→ 구매할까?
→ NO!

마찬가지로 우리도 미국처럼 보여야 함
```

### 현지화 영역

```
1. 통화 (Currency)
2. 위치 (Location)
3. 연락처 (Phone)
4. 이메일 (Email)
5. 주소 (Address)
6. 배송비 (Shipping)
7. 시간대 (Timezone)
```

---

## 2. 통화/마켓 설정

### Shopify 설정

```
Settings → Markets → United States

[설정 항목]
- Currency: USD ($)
- Language: English
- Domain: .com (권장)
```

### 왜 중요한가?

```
원화로 가격 표시되면:
- 미국인: "이거 뭔 돈이야?"
- 거부감 → 이탈

달러로 표시:
- 익숙함 → 신뢰 → 구매
```

## 3. 전화번호 설정

### OpenPhone 추천

```
https://www.openphone.com

[특징]
- 미국 전화번호 발급
- 앱으로 관리
- 문자 수신 가능
- 월 $15-20
```

### 지역번호 선택

| 지역번호 | 지역 | 특징 |
|----------|------|------|
| **310** | LA (로스앤젤레스) | 일반 소비자 |
| **212** | 뉴욕 | 비즈니스 |
| **408** | 실리콘밸리 | 테크 |
| **305** | 마이애미 | 라틴 |
| **213** | LA 다운타운 | 일반 |

### 설정 과정

```
1. OpenPhone 회원가입
2. 지역 선택 (예: Los Angeles)
3. 번호 선택 (310-XXX-XXXX)
4. 결제 등록
5. 앱 설치
6. Shopify에 번호 입력
```

## 4. 이메일 설정

### 권장 형식

```
[비즈니스 이메일]
support@yourstorename.com
hello@yourstorename.com
contact@yourstorename.com

[간단 대안]
yourstorename@gmail.com
```

### Gmail 설정

```
1. Gmail 새 계정 생성
2. 샵 이름으로 계정명 설정
3. 프로필 사진 = 샵 로고
4. Shopify에 연결
```

### 이메일 관리

```
5개 샵 = 5개 이메일?

효율적 방법:
- Google 계정 2-3개
- 각 계정에 샵 2-3개
- 라벨로 구분
```

## 5. 배송비 설정 ⭐

### 설정 위치

```
Settings → Shipping and delivery → Shipping
```

### 기존 설정 삭제

```
1. 기본 배송 옵션 모두 삭제
2. Add rate 클릭
3. 새로 설정
```

### 권장 설정

```
[배송 옵션]
Name: Standard Shipping
Price: $7.99 (고정)
Delivery time: 5-8 business days

→ 상품 5만개 일괄 적용
→ 무료 배송이어도 $7.99 = 마진
```

### 무료 배송 vs 유료

| 옵션 | 장점 | 단점 |
|------|------|------|
| **무료** | 전환율 높음 | 마진 감소 |
| **$4.99** | 적당한 마진 | 전환율 약간 감소 |
| **$7.99** | 좋은 마진 | 일부 이탈 |
| **$9.99+** | 높은 마진 | 이탈 증가 |

### 미국 배송비 인식

```
미국은 땅이 넓음:
- 뉴욕 → LA = $14+ 기본
- 배송비 비싼 게 일상
- $7.99는 저렴한 편

→ 미국인에게 배송비 부담 적음
```

### 인터내셔널 제거

```
Settings → Shipping → International

삭제 권장:
- 유럽, 호주 등
- 배송비 너무 비쌈
- 현실적으로 불가

→ 미국만 남기기
```

## 6. 주소 설정

### 비즈니스 주소

```
[옵션 1] 가상 오피스
- 미국 주소 제공
- 월 $10-50
- 우편물 수령 가능

[옵션 2] 실제 주소
- 미국 법인 설립 시
- 비용 높음
- 장기적 목표
```

### GSK 서비스 활용

```
GSK 정산 서비스 이용 시:
- 미국 주소 제공
- 비즈니스 인프라 지원
- 초기 비용 절감
```

## 7. 시간대 설정

### Shopify 시간대

```
Settings → General → Timezone

권장: (GMT-08:00) Pacific Time (US & Canada)
또는: (GMT-05:00) Eastern Time (US & Canada)

→ 미국 시간 기준 운영
→ 리포트/통계 일관성
```

## 8. 현지화 체크리스트

### 필수 항목

```
[통화/마켓]
□ Currency = USD
□ Language = English
□ Market = United States

[연락처]
□ 미국 전화번호 (OpenPhone)
□ 비즈니스 이메일
□ Contact 페이지에 표시

[배송]
□ 기존 배송 옵션 삭제
□ Standard $7.99 설정
□ 5-8 days 표시
□ International 제거

[시간대]
□ Pacific 또는 Eastern Time

[정책]
□ Return Policy
□ Shipping Policy
□ Privacy Policy
□ Terms of Service
```

### 선택 항목

```
[고급]
□ 미국 비즈니스 주소
□ 도메인 .com
□ SSL 인증서 (자동)
□ 미국 결제 게이트웨이
```

## 9. 다중 샵 현지화

### 효율적 설정

```
[첫 샵]
- 꼼꼼하게 설정
- 모든 항목 확인
- 템플릿으로 저장

[2번째 샵부터]
- 첫 샵 설정 복사
- 이름/이메일만 변경
- 빠르게 완료
```

### 계정 분리

```
Google 계정 1: 샵 A, B, C
Google 계정 2: 샵 D, E, F

→ 리스크 분산
→ 관리 편의성
```
