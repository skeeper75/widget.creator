# Shopify Global Commerce Skill

Shopify 기반 미국 시장 드랍쉬핑 비즈니스 운영 가이드.
스파미 오프라인 강의(2025.12) 실전 노하우 반영.

## Quick Reference

### 마진 계산 공식
```
실수령액 = 결제금액 × 0.971 - $0.30
마진 = 실수령액 - 상품원가 - 배송비
마진율 = (마진 ÷ 결제금액) × 100
```

### Shopify 수수료 구조
| 항목 | 비율/금액 |
|------|-----------|
| 카드 결제 수수료 | 2.9% |
| 건당 고정 수수료 | $0.30 |
| PayPal 등 외부 결제 | 추가 0.5~2% |

### 핵심 경고 사항 ⚠️
```
1. LA = Louisiana (로스앤젤레스 X!)
2. 플로리다 창고 = 차지백 고위험
3. 구독구매 함정 = One-time Purchase 필수
4. IP 신고 = 3일 내 응답 필수
5. 하나에 매몰 X = 옆으로 확장
```

## Workflows

### 1. 샵 초기 설정 체크리스트

미국 시장 타겟팅을 위한 필수 설정:

```
□ Settings > 일반 > 스토어 세부정보 → 미국 주소 설정
□ Settings > 일반 > 스토어 기본설정 → USD 통화 설정  
□ Settings > 위치 → 미국 위치 설정
□ Settings > 마켓 → United States 추가 및 활성화
□ Settings > 배송 및 배달 → $7.99 고정, International 삭제
□ 전화번호 → OpenPhone으로 미국 번호 발급 (310 권장)
□ 제품 최소 1개 등록 (마켓 활성화 필수조건)
```

### 2. 주소 검증 5초 체크

발주 전 필수 확인:

```
□ 주 코드 + 우편번호 첫자리 일치?
  - CA = 9로 시작
  - LA = Louisiana (7로 시작) ← 로스앤젤레스 아님!
□ 구글맵에서 빈 공간? → ChatGPT로 확인
□ 푸에르토리코(PR)? → 캔슬 금지, 정상 배송
□ 창고/웨어하우스 주소? → 차지백 위험
□ 플로리다 + 창고? → 캔슬 고려
```

### 3. 차지백 방어 전략

```
[위험 신호]
- 빨간불 경고
- 카드 여러 장 시도
- 플로리다 + 창고 주소
- $200+ 고가 상품

[저가 = 안전]
- $50 이하 = 사기꾼도 타산 안 맞음
- 홈페이지 신뢰도 = 최고의 방어

[소명 증거]
- 배송 완료 사진 (있으면 확정 승소)
- 고객 소통 없이 바로 차지백 = 사기 증거
```

### 4. CS 핵심 원칙

```
[미국 이메일 문화]
- 공감 먼저, 해결 나중
- "We understand how frustrating..."

[대문자 = 화남]
- "WHERE IS MY ORDER?" = 소리지름
- 즉시 대응 필요

[협상 가능]
- 품절 → 부분 취소 제안 (하나라도 팔기)
- 색상 품절 → 대체 색상 제안
- 세트 품절 → 개별 2개로 대체
```

### 5. 발주 체크리스트

```
□ 사기 경고 확인
□ 주소 검증 (주+우편번호)
□ 재고/가격 확인
□ One-time Purchase 선택 ← 구독 금지!
□ 트래킹 입력 + 고객 알림
```

### 6. 샵 확장 전략

```
[목표]
- 미니멈 5개
- 내년 목표 10개
- 전업 시 50개까지

[원칙]
- 하나에 매몰 금지
- 남는 시간 = 새 샵 만들기
- 예비 샵 2-3개 준비
```
