# 디자인 시스템

huni.builder의 디자인 시스템은 Atomic Design 패턴을 기반으로 구축되었으며, 일관된 UI/UX를 제공하기 위해 디자인 토큰, 컴포넌트 라이브러리, 스타일 가이드라인을 체계적으로 관리합니다.

## 개요

디자인 시스템은 다음과 같은 핵심 요소로 구성됩니다:

- **디자인 토큰**: 색상, 타이포그래피, 스페이싱, 보더 레디우스 등의 시스템 전반 재사용 가능한 값
- **Atomic Design**: Atom, Molecule, Organism 계층 구조로 구성된 컴포넌트 라이브러리
- **Shadow DOM 격리**: 위젯이 호스트 페이지의 스타일과 독립적으로 동작하도록 보장
- **CSS Variables**: 테마 커스터마이징과 다크/라이트 모드 지원

## Shadow DOM 스타일링

huni.builder 위젯은 Shadow DOM을 사용하여 호스트 페이지의 스타일과 완전히 격리됩니다.

```typescript
// 위젯 초기화 시 Shadow DOM 생성
const shadowRoot = this.attachShadow({ mode: 'open' })

// CSS Variables 주입을 통한 테마 커스터마이징
const sheet = new CSSStyleSheet()
sheet.replaceSync(`
  :host {
    --hw-color-primary: #5538b6;
    --hw-color-accent: #f0831e;
    /* ... */
  }
`)
shadowRoot.adoptedStyleSheets = [sheet]
```

## 테마 커스터마이징

CSS Variables를 사용하여 위젯의 외형을 커스터마이징할 수 있습니다:

```css
/* 호스트 페이지에서 위젯 테마 재정의 */
huni-widget {
  --hw-color-primary: #your-brand-color;
  --hw-font-family: 'Your Font', sans-serif;
  --hw-radius-md: 8px;
}
```
