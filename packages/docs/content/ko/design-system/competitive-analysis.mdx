# 경쟁사 UI 비교

huni.builder와 경쟁사(와우프레스, 바이상상)의 UI/UX 차이점과 huni.builder의 차별화된 장점을 분석합니다.

## 와우프레스 (Wowpress)

### UI 패턴

- **카테고리 드롭다운**: 상단 드롭다운으로 카테고리 선택
- **순차 선택**: 카테고리 → 옵션 → 수량 순서로 진행
- **실시간 가격 표시**: 옵션 선택 시 즉시 가격 반영

### 장점

- 익숙한 드롭다운 패턴
- 간단한 흐름
- 빠른 옵션 선택

### 한계

- 카테고리 많을 경우 드롭다운이 복잡해짐
- 이전 단계로 돌아가기 불편
- 한 화면에서 모든 옵션을 볼 수 없음

### UI 예시

```
┌─────────────────────────────────────┐
│ 카테고리 ▼ [엽서] [스티커] [포스터]  │
├─────────────────────────────────────┤
│ 용지 선택                           │
│ ○ 아트지  ○ 모조지  ○ 스노우지     │
│                                     │
│ 규격 선택                           │
│ ○ A4  ○ A5  ○ B5                   │
├─────────────────────────────────────┤
│ 수량: [100] | 가격: \10,000         │
└─────────────────────────────────────┘
```

## 바이상상 (Buysangsang/Bizhows)

### UI 패턴

- **칩/버튼 기반 옵션 선택**: 카드 형태의 옵션 선택
- **장당 단가 표시**: 개당 가격 표시
- **조건부 활성화**: 선행 옵션에 따른 후속 옵션 활성화

### 장점

- 시각적으로 명확한 옵션 표시
- 가격 투명성
- 조건부 로직에 따른 UX 가이드

### 한계

- 옵션이 많을 경우 스크롤이 길어짐
- 작은 화면에서 카드 레이아웃이 압박감
- 전체 진행 상황 파악 어려움

### UI 예시

```
┌─────────────────────────────────────┐
│ 1. 용지 선택                        │
│ ┌─────────┐ ┌─────────┐ ┌─────────┐ │
│ │ 아트지   │ │ 모조지   │ │ 스노우지 │ │
│ │ +100원  │ │ +150원  │ │ +200원  │ │
│ └─────────┘ └─────────┘ └─────────┘ │
│                                     │
│ 2. 규격 선택 (활성화됨)             │
│ ┌─────┐ ┌─────┐ ┌─────┐            │
│ │ A4  │ │ A5  │ │ B5  │            │
│ └─────┘ └─────┘ └─────┘            │
└─────────────────────────────────────┘
```

## huni.builder

### UI 패턴

- **위자드 방식**: 단계별 옵션 선택과 진행 상황 표시
- **Shadow DOM 격리**: 호스트 페이지 스타일과 독립
- **디자인 토큰 기반**: 일관된 UI와 테마 커스터마이징
- **반응형 레이아웃**: 모든 화면 크기 최적화

### 장점

- **진행 상황 명확**: StepIndicator로 현재 위치 표시
- **스타일 격리**: 호스트 페이지 스타일 영향 없음
- **테마 커스터마이징**: CSS Variables로 쉬운 테마 변경
- **접근성**: WCAG 2.1 AA 준수
- **확장성**: Atomic Design으로 재사용 가능한 컴포넌트

### UI 예시

```
┌─────────────────────────────────────┐
│ ●━━━━○━━━━○━━━━○  3단계 중 1단계   │
├─────────────────────────────────────┤
│ 용지 선택                           │
│                                     │
│ ┌─────────────────────┐             │
│ │ ◉ 아트지            │             │
│ │ 일반적인 용지       │             │
│ │ +100원 / 매수      │             │
│ └─────────────────────┘             │
│                                     │
│ ┌─────────────────────┐             │
│ │ ○ 모조지            │             │
│ │ 부드러운 질감       │             │
│ │ +150원 / 매수      │             │
│ └─────────────────────┘             │
├─────────────────────────────────────┤
│      [이전]  [다음]                 │
│      현재 가격: \10,000              │
└─────────────────────────────────────┘
```

## 기능 비교표

| 기능 | 와우프레스 | 바이상상 | huni.builder |
|------|-----------|----------|--------------|
| 카테고리 선택 | 드롭다운 | 칩 | 위자드 단계 |
| 옵션 선택 | 라디오/셀렉트 | 카드 | 카드 + 위자드 |
| 진행 상황 | 없음 | 약간 | StepIndicator |
| 가격 표시 | 실시간 | 장당 단가 | 실시간 + 요약 |
| 스타일 격리 | 없음 | 없음 | Shadow DOM |
| 테마 변경 | 불가능 | 불가능 | CSS Variables |
| 반응형 | 부분 | 부분 | 완전 |
| 접근성 | 기본 | 기본 | WCAG 2.1 AA |
| 디자인 시스템 | 없음 | 없음 | Atomic Design |

## 개선 방향

### 현재 huni.builder의 강점

1. **위자드 방식**: 복잡한 옵션을 단계적으로 안내
2. **Shadow DOM**: 호스트 페이지와 완전한 스타일 격리
3. **디자인 토큰**: 일관된 UI와 테마 커스터마이징
4. **접근성**: 키보드 내비게이션, 스크린 리더 지원

### 추가 개선 포인트

1. **빠른 옵션 선택**: 자주 사용하는 조합 빠른 선택
2. **옵션 미리보기**: 선택한 옵션의 시각적 미리보기
3. **가격 범위 표시**: 수량별 가격 변화 그래프
4. **최근 선택 기억**: 재방문 시 이전 선택 복원

### 제안된 개선 UI

```
┌─────────────────────────────────────┐
│ ●━━━━○━━━━○━━━━○  3단계 중 1단계   │
├─────────────────────────────────────┤
│ [빠른 선택] [최근 선택] [맞춤 설정] │
│                                     │
│ 용지 선택                           │
│ ┌─────────────────────┐             │
│ │ ◉ 아트지    [미리보기]│            │
│ │ 일반적인 용지       │             │
│ │ 100매: \10,000      │             │
│ │ 500매: \45,000      │             │
│ │ 1000매: \80,000     │             │
│ │ [▼ 가격 그래프]     │             │
│ └─────────────────────┘             │
└─────────────────────────────────────┘
```

## 기술적 차별점

### Shadow DOM 격리

```typescript
// huni.builder만 제공하는 완전한 스타일 격리
const shadowRoot = this.attachShadow({ mode: 'open' })
// 호스트 페이지 CSS 영향 없음
```

### CSS Variables 테마

```css
/* 테마 커스터마이징 */
huni-widget {
  --hw-color-primary: #your-brand;
  --hw-radius-md: 10px;
}
```

### Atomic Design

```
Atoms → Molecules → Organisms → Templates → Pages
재사용 가능한 컴포넌트 계층 구조
```
