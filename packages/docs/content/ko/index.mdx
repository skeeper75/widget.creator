# huni.builder 문서

**huni.builder**는 이커머스 플랫폼을 위한 가격 위젯을 빌드하고 관리하는 플랫폼입니다.

## 주요 기능

- **위젯 코어**: 가격 표시 위젯 컴포넌트
- **가격 엔진**: 유연한 가격 계산 시스템
- **관리자 대시보드**: 위젯 설정 및 관리
- **API 서버**: RESTful API 백엔드
