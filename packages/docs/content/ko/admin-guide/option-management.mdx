# 옵션 관리

10개 옵션 표시 타입과 옵션 카드 관리 방법을 설명합니다.

> **핵심**: 30개 옵션 키를 10가지 다른 UI 컴포넌트로 표시할 수 있습니다.

<Callout type="info">
**드래그앤드롭**: 옵션 순서는 드래그앤드롭으로 쉽게 변경할 수 있습니다.
</Callout>

## 1. 옵션 표시 타입 (10개)

| 타입 | 한글 라벨 | UI 컴포넌트 | 설명 | 예시 |
|------|-----------|-------------|------|------|
| button-group | 버튼 그룹 | 버튼 칩 | 단일 선택 (버튼) | Size 선택 |
| select-box | 드롭다운 | 드롭다운 | 단일 선택 (리스트) | Paper type |
| count-input | 숫자 입력 | 숫자 스테퍼 | 숫자 값 (min/max/step) | Page count |
| color-chip | 칼라 칩 | 컬러 스와치 | 컬러 선택 (시각적) | Foil color |
| image-chip | 이미지 칩 | 이미지 카드 | 이미지 기반 선택 | Accessory preview |
| toggle-group | 토글 그룹 | 토글 스위치 | 복수 불리언 토글 | Finishing options |
| finish-button | 후가공 버튼 | 후가공 버튼 | 후가공 선택 | Coating type |
| finish-title-bar | 후가공 그룹 | 그룹 헤더 | 후가공 카테고리 헤더 | Foil stamp group |
| summary | 요약 표시 | 읽기 전용 | 현재 선택 요약 | Order summary |
| upload | 파일 업로드 | 파일 입력 | 커스텀 아트웍 업로드 | Design file |

## 2. 옵션 카드 해부

```
┌─────────────────────────────────────────────┐
│  ●  Size                         [≡] [⨉]    │
│  단일 선택 / select-box                     │
└─────────────────────────────────────────────┘
```

**아이콘 범례**:

| 아이콘 | 색상 | 의미 |
|--------|-------|---------|
| ● (채워진 원) | 빨강 | 필수 옵션 |
| ₩ (원 기호) | 녹색 | 가격 규칙 연결 |
| ! (느낌표) | 주황 | 제약조건 있음 |
| → (오른쪽 화살표) | 파랑 | 다른 옵션 트리거 |
| ← (왼쪽 화살표) | 보라 | 다른 옵션 의존 |
| ⋮⋮ (그립 점) | 회색 | 드래그 핸들 |

## 3. 옵션 추가/편집

<Steps>
### Step 1: [+ Add Option] 클릭

원하는 레이어에서 클릭

### Step 2: 옵션 폼 작성

- **이름**: 표시 이름
- **키 (영문)**: 옵션 키
- **타입**: 표시 타입 (10개 중 하나)
- **필수**: 필수 여부
- **레이어**: 레이어 할당
- **기본값**: 기본 선택값

### Step 3: count-input 타입 설정

min, max, step 값 설정

### Step 4: select-box 타입 설정

선택지 목록 관리

### Step 5: 저장 및 확인
</Steps>

## 4. 선택지 관리

### 4.1 선택지 추가

- 수동 입력: 하나씩 추가
- CSV import: 대량 추가
- 붙여넣기: 대량 붙여넣기

### 4.2 선택지 속성

| 속성 | 설명 |
|------|------|
| label | 표시 이름 |
| value | 내부 값 |
| priceKey | 가격 키 |
| code | 코드 |
| specs | 사양 |

### 4.3 선택지 재정렬

드래그앤드롭으로 선택지 순서 변경

<Callout type="info">
**저장**: 변경 사항은 자동으로 저장됩니다 (optimistic updates).
</Callout>

## 5. 계층적 옵션 트리

```
option_templates (옵션 템플릿)
    ↓
product_options (상품 옵션)
    ↓
option_choices (옵션 선택지)
```

- **Templates**: per-product_type 기본값 제공
- **Product Options**: 템플릿을 상품별로 커스터마이즈
- **Option Choices**: 선택 가능한 값 정의

## 6. 드래그앤드롭 정렬

### 6.1 레이어 내 재정렬

드래그 핸들로 위/아래 이동

### 6.2 레이어 간 이동

옵션 카드를 다른 레이어 섹션으로 드래그

### 6.3 키보드 단축키

- Alt+Up: 위로 이동
- Alt+Down: 아래로 이동

### 6.4 시각적 피드백

파란색 드롭 표시기 라인

## 7. 종속 옵션과 제약조건

### 7.1 시각적 표시기

옵션 카드의 화살표 아이콘:
- → (오른쪽 화살표): 다른 옵션 트리거
- ← (왼쪽 화살표): 다른 옵션 의존

### 7.2 트리거/의존성 동작

옵션 선택이 다른 옵션의 가용성에 영향

### 7.3 제약조건 설정

[제약조건 에디터](./constraints) 페이지에서 상세 관리

### 7.4 EdicusOptionManager

Edicus 온라인 디자인 편집기 통합 (커스텀 아트웍 편집)
