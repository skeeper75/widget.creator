# huni.builder Documentation

**huni.builder** is a platform for building and managing pricing widgets for e-commerce platforms.

## Key Features

- **Widget Core**: Pricing display widget components
- **Pricing Engine**: Flexible pricing calculation system
- **Admin Dashboard**: Widget configuration and management
- **API Server**: RESTful API backend

## Quick Start

Get started with huni.builder by following the [Getting Started Guide](/en/getting-started) to install and create your first widget.
