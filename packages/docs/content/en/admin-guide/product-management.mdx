## 2. Creating a New Product

<Steps>
### Step 1: Click [+ Product]

Or right-click category > Add Product

### Step 2: Fill Product Header

- **Name**: Product name (e.g., A4 Postcard)
- **MES Code**: `XXX-XXXX` format (e.g., `001-0001`)
- **Huni Code**: Product identifier code
- **Product Type**: Category-specific type

### Step 3: Select Category

Choose category from tree

### Step 4: Set Initial Status

Active/Inactive toggle

### Step 5: Choose Category Group Template

Select from G1-G8 (auto-configured)
</Steps>

<Callout type="info">
**MES Code Format**: `001-0001` format, first 3 digits = category, last 4 digits = product number.
</Callout>

<Callout type="warning">
**Category Assignment**: Category determines available option templates. Choose carefully.
</Callout>

### 3.4 PropertyPanel (Right Panel)

Configuration for selected component:

**Common fields**:
- label: Display name
- optionKey: Option key (English)
- optionType: Display type
- isRequired: Required status
- displayOrder: Display order

**Type-specific fields**:
- SizeSelector: allowCustom, unitType, minWidth/maxWidth
- PaperSelector: paperGroups, ncrMode, showGrammage
- QuantitySelector: quantityMode, showUnitPrice

### 3.5 TemplateSelector

Apply category group defaults (G1-G8)

| Group | Name | Default Components |
|-------|------|-------------------|
| G1 | Flat Prints | Size, Color, Paper, Qty, PostProcess, Delivery |
| G2 | Stickers | Shape, Size, Material, Color, Qty, DieCut, Delivery |
| G3 | Booklets | CoverType, Size, Paper, Color, Qty, PostProcess, Delivery |
| G4 | Forms (NCR) | NCRType, Size, Paper(1-4), Color, Qty, Numbering, Delivery |
| G5 | Large Signs | Material, Width(cm), Height(cm), Color, Qty, Fixtures, Delivery |
| G6 | Goods/Fancy | Product, Size, Color, Print, Qty, Package, AddOns, Delivery |
| G7 | Digital Print | Printer, Size, Paper, Color, Qty, Finishing, Delivery |
| G8 | Apparel | Size(chart), Color(swatch), PrintPos, PrintSize, Qty, Delivery |

<Callout type="info">
**Save**: Changes are saved automatically (optimistic updates).
</Callout>

<Callout type="warning">
**Warning**: Deletion is irreversible. Backup before deleting.
</Callout>

**Cascading effects**:
- Linked options are also deleted
- Constraint rules are also deleted
- Price rules are also deleted
